# mysql-docs

# Etapas para Formatação

1. Use o script `./scripts/move_images.py`
2. Use o script `./scripts/format_md.py` para arquivos que estão no início do diretório
3. Use o script `./scripts/format_dir.py` referente aos diretórios dos capítulos
4. Use o script `./scripts/join_files.py` para juntar os subdiretórios de cada pasta-capítulo
5. Use o script `./scripts/remove_toc.py` para remover a tabela de conteúdo no início dos arquivos dentro dos capítulos
