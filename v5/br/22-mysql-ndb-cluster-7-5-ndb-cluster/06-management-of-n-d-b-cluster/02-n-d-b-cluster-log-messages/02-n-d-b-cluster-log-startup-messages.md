#### 21.6.2.2 Mensagens de Inicialização do Log do NDB Cluster

Mensagens de inicialização possíveis com descrições são fornecidas na lista a seguir:

* `Initial start, waiting for %s to connect, nodes [ all: %s connected: %s no-wait: %s ]`
  * Início inicial, esperando que %s se conecte, nodes [ todos: %s conectados: %s no-wait: %s ]

* `Waiting until nodes: %s connects, nodes [ all: %s connected: %s no-wait: %s ]`
  * Esperando até que os nodes: %s se conectem, nodes [ todos: %s conectados: %s no-wait: %s ]

* `Waiting %u sec for nodes %s to connect, nodes [ all: %s connected: %s no-wait: %s ]`
  * Esperando %u segundos pelos nodes %s se conectarem, nodes [ todos: %s conectados: %s no-wait: %s ]

* `Waiting for non partitioned start, nodes [ all: %s connected: %s missing: %s no-wait: %s ]`
  * Esperando por start não particionado, nodes [ todos: %s conectados: %s faltando: %s no-wait: %s ]

* `Waiting %u sec for non partitioned start, nodes [ all: %s connected: %s missing: %s no-wait: %s ]`
  * Esperando %u segundos por start não particionado, nodes [ todos: %s conectados: %s faltando: %s no-wait: %s ]

* `Initial start with nodes %s [ missing: %s no-wait: %s ]`
  * Initial start com nodes %s [ faltando: %s no-wait: %s ]

* `Start with all nodes %s`
  * Start com todos os nodes %s

* `Start with nodes %s [ missing: %s no-wait: %s ]`
  * Start com nodes %s [ faltando: %s no-wait: %s ]

* `Start potentially partitioned with nodes %s [ missing: %s no-wait: %s ]`
  * Start potencialmente particionado com nodes %s [ faltando: %s no-wait: %s ]

* `Unknown startreport: 0x%x [ %s %s %s %s ]`
  * startreport desconhecido: 0x%x [ %s %s %s %s ]
