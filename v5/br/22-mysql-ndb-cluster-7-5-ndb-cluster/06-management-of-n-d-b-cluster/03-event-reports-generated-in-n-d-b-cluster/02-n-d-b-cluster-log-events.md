#### 21.6.3.2 Eventos de registro do cluster NDB

Um relatório de evento registrado nos logs de evento tem o seguinte formato:

```sql
datetime [string] severity -- message
```

Por exemplo:

```sql
09:19:30 2005-07-24 [NDB] INFO -- Node 4 Start phase 4 completed
```

Esta seção discute todos os eventos que devem ser relatados, ordenados por categoria e nível de gravidade dentro de cada categoria.

Nas descrições de eventos, GCP e LCP significam “Ponto de Controle Global” e “Ponto de Controle Local”, respectivamente.
