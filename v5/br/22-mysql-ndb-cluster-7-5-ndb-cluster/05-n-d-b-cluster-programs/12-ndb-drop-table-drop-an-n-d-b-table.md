### 21.5.12 ndb_drop_table — Drop an NDB Table

[**ndb_drop_table**](mysql-cluster-programs-ndb-drop-table.html "21.5.12 ndb_drop_table — Drop an NDB Table") drops the specified [`NDB`](mysql-cluster.html "Chapter 21 MySQL NDB Cluster 7.5 and NDB Cluster 7.6") table. (If you try to use this on a table created with a storage engine other than [`NDB`](mysql-cluster.html "Chapter 21 MySQL NDB Cluster 7.5 and NDB Cluster 7.6"), the attempt fails with the error 723: No such table exists.) This operation is extremely fast; in some cases, it can be an order of magnitude faster than using a MySQL [`DROP TABLE`](drop-table.html "13.1.29 DROP TABLE Statement") statement on an [`NDB`](mysql-cluster.html "Chapter 21 MySQL NDB Cluster 7.5 and NDB Cluster 7.6") table.

#### Usage

```sql
ndb_drop_table -c connection_string tbl_name -d db_name
```
