Esta seção descreve os procedimentos de instalação para o NDB Cluster em hosts Windows. Os binários do NDB Cluster 7.5 e 7.6 para Windows podem ser obtidos em <https://dev.mysql.com/downloads/cluster/>. Para obter informações sobre a instalação do NDB Cluster no Windows a partir de um binário fornecido pela Oracle, consulte [Section 21.3.2.1, “Instalando o NDB Cluster no Windows a partir de um Binário”](mysql-cluster-install-windows-binary.html "21.3.2.1 Instalando o NDB Cluster no Windows a partir de um Binário").

Também é possível compilar e instalar o NDB Cluster a partir do código-fonte no Windows usando o Microsoft Visual Studio. Para mais informações, consulte [Section 21.3.2.2, “Compilando e Instalando o NDB Cluster a partir do Código-Fonte no Windows”](mysql-cluster-install-windows-source.html "21.3.2.2 Compilando e Instalando o NDB Cluster a partir do Código-Fonte no Windows").
