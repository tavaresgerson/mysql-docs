Esta seção descreve os procedimentos de instalação do NDB Cluster em hosts Windows. Os binários do NDB Cluster 7.5 e 7.6 para Windows podem ser obtidos em <https://dev.mysql.com/downloads/cluster/>. Para obter informações sobre a instalação do NDB Cluster em Windows a partir de uma versão binária fornecida pela Oracle, consulte Seção 21.3.2.1, “Instalando o NDB Cluster em Windows a partir de uma Versão Binária”.

É também possível compilar e instalar o NDB Cluster a partir da fonte no Windows usando o Microsoft Visual Studio. Para mais informações, consulte Seção 21.3.2.2, “Compilar e Instalar o NDB Cluster a partir da Fonte no Windows”.
