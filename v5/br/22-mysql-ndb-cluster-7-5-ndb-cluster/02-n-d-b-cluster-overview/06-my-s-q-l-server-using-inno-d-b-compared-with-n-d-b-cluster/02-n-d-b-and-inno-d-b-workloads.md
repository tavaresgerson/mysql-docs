#### 21.2.6.2 Cargas de trabalho NDB e InnoDB

O NDB Cluster possui uma série de atributos únicos que o tornam ideal para atender a aplicações que exigem alta disponibilidade, failover rápido, alto desempenho e baixa latência. Devido à sua arquitetura distribuída e implementação de vários nós, o NDB Cluster também tem restrições específicas que podem impedir que algumas cargas de trabalho funcionem bem.
