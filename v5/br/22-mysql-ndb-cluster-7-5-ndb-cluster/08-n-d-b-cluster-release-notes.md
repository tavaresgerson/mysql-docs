## 21.8 Notas de Lançamento do NDB Cluster

As alterações nas versões do NDB Cluster são documentadas separadamente deste manual de referência; você pode encontrar as notas de lançamento para as alterações em cada versão do NDB Cluster 7.5 em [Notas de Lançamento do NDB 7.5](/doc/relnotes/mysql-cluster/7.5/en/), e para cada versão do NDB Cluster 7.6 em [Notas de Lançamento do NDB 7.6](/doc/relnotes/mysql-cluster/7.6/en/).

Você pode obter as notas de lançamento para versões mais antigas do NDB Cluster em [Notas de Lançamento do NDB Cluster](/doc/index-cluster.html#cluster-relnotes).
