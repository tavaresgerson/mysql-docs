### 7.4.5 Dicas do mysqldump

Esta seção aborda técnicas que permitem usar o **mysqldump** para resolver problemas específicos:

- Como fazer uma cópia de um banco de dados
- Como copiar um banco de dados de um servidor para outro
- Como descartar programas armazenados (procedimentos e funções armazenados, gatilhos e eventos)
- Como descartar definições e dados separadamente
