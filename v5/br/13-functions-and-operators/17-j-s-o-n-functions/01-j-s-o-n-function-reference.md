O MySQL 5.7.22 e posterior suporta duas funções JSON de agregação, `JSON_ARRAYAGG()` e `JSON_OBJECTAGG()`. Consulte a Seção 12.19, “Funções de Agregação”, para descrições destas.

Também a partir do MySQL 5.7.22:

* A "impressão formatada" (*pretty-printing*) de valores JSON em um formato fácil de ler pode ser obtida usando a função `JSON_PRETTY()`.

* É possível verificar quanto espaço de armazenamento um determinado valor JSON ocupa usando `JSON_STORAGE_SIZE()`.

Para descrições completas dessas duas funções, consulte a Seção 12.17.6, “Funções Utilitárias JSON”.
