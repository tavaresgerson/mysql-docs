O MySQL 5.7.22 e versões posteriores suportam duas funções agregadas JSON `JSON_ARRAYAGG()` e `JSON_OBJECTAGG()`. Consulte a Seção 12.19, “Funções Agregadas”, para obter descrições dessas funções.

Também a partir do MySQL 5.7.22:

- A “imprimação bonita” dos valores JSON em um formato fácil de ler pode ser obtida usando a função `JSON_PRETTY()`.

- Você pode ver quanto espaço de armazenamento um determinado valor JSON ocupa usando `JSON_STORAGE_SIZE()`.

Para descrições completas dessas duas funções, consulte a Seção 12.17.6, “Funções de Utilitário JSON”.
