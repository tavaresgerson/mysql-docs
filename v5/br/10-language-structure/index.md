# Capítulo 9 Estrutura da Linguagem

Este capítulo discute as regras para escrever os seguintes elementos das instruções SQL ao usar o MySQL:

- Valores literais, como strings e números
- Identificadores como nomes de banco de dados, tabelas e colunas
- Palavras-chave e palavras reservadas
- Variáveis definidas pelo usuário e variáveis do sistema
- Expressões
- Comentários
