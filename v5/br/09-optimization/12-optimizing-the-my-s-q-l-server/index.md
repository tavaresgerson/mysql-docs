## 8.12. Otimizando o servidor MySQL

Esta seção discute técnicas de otimização para o servidor de banco de dados, focando principalmente na configuração do sistema, em vez de no ajuste de instruções SQL. As informações nesta seção são apropriadas para DBAs que desejam garantir o desempenho e a escalabilidade em todos os servidores que gerenciam; para desenvolvedores que constroem scripts de instalação que incluem a configuração do banco de dados; e para pessoas que executam o MySQL por si mesmas para desenvolvimento, testes, etc., que desejam maximizar sua própria produtividade.
