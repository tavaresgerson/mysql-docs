8.4.3.2 Desvantagens de criar muitas tabelas na mesma base de dados

Algumas técnicas para manter as consultas individuais rápidas envolvem a divisão dos dados em várias tabelas. Quando o número de tabelas chega a milhares ou até milhões, o custo de lidar com todas essas tabelas se torna uma nova consideração de desempenho.
