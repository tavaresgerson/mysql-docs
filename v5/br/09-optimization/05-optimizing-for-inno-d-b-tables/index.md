`InnoDB` é o mecanismo de armazenamento que os clientes do MySQL normalmente usam em bancos de dados de produção onde a confiabilidade e a concorrência são importantes. `InnoDB` é o mecanismo de armazenamento padrão do MySQL. Esta seção explica como otimizar as operações do banco de dados para tabelas `InnoDB`.
