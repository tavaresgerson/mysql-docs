#### 5.1.8.2 Variáveis dinâmicas do sistema

Muitas variáveis do sistema do servidor são dinâmicas e podem ser definidas em tempo de execução. Consulte Seção 13.7.4.1, “Sintaxe de definição de variáveis”. Para uma descrição dos requisitos de privilégio para definir variáveis do sistema, consulte Seção 5.1.8.1, “Privilégios de variáveis do sistema”

A tabela a seguir lista todas as variáveis de sistema dinâmicas aplicáveis no `mysqld`.

A tabela lista o tipo de dado e o escopo de cada variável. A última coluna indica se o escopo de cada variável é Global, Sessão ou ambos. Consulte as descrições dos itens correspondentes para obter detalhes sobre a configuração e o uso das variáveis. Quando aplicável, são fornecidos links diretos para informações adicionais sobre os itens.

Variáveis com tipo "string" aceitam um valor de string. Variáveis com tipo "numeric" aceitam um valor numérico. Variáveis com tipo "booleano" podem ser definidas como 0, 1, `ON` ou `OFF`. Variáveis marcadas como "enumeração" normalmente devem ser definidas para um dos valores disponíveis para a variável, mas também podem ser definidas para o número que corresponde ao valor de enumeração desejado. Para variáveis de sistema enumeradas, o primeiro valor de enumeração corresponde a 0. Isso difere do tipo de dados `ENUM` usado para colunas de tabelas, para o qual o primeiro valor de enumeração corresponde a 1.
