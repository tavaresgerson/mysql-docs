## 27.11 APIs Ruby para MySQL

[27.11.1 A API MySQL/Ruby](apis-ruby-mysqlruby.html)

[27.11.2 A API Ruby/MySQL](apis-ruby-rubymysql.html)

A `gem` Ruby `mysql2` fornece uma API para conectar-se ao MySQL, executar `queries` e iterar pelos resultados; ela é projetada para oferecer suporte ao MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página `mysql2` em RubyGems.org ou a página do `GitHub` do projeto.

Para informações de contexto e sintaxe sobre a linguagem Ruby, consulte [Ruby Programming Language](http://www.ruby-lang.org).
