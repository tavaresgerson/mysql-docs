27.11.1 API MySQL/Ruby

27.11.1 API Ruby/MySQL

O gem Ruby `mysql2` fornece uma API para conectar-se ao MySQL, realizar consultas e iterar pelos resultados; ele é projetado para suportar MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página do `mysql2` no RubyGems.org (<https://rubygems.org/gems/mysql2/>) ou a página do projeto no GitHub (<https://github.com/brianmario/mysql2>).

Para informações sobre o contexto e a sintaxe da linguagem Ruby, consulte Ruby Programming Language.
