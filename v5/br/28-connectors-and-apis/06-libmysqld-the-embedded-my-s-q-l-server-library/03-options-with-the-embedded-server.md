### 27.6.3 Opções com o Servidor Integrado

Quaisquer opções que possam ser fornecidas com o daemon do servidor **mysqld** podem ser usadas com uma biblioteca de servidor embutida. As opções do servidor podem ser fornecidas em um array como argumento para a função `mysql_library_init()`, que inicializa o servidor. Elas também podem ser fornecidas em um arquivo de opções como `my.cnf`. Para especificar um arquivo de opções para um programa C, use a opção `--defaults-file` como um dos elementos do segundo argumento da função `mysql_library_init()`. Veja mysql\_library\_init() para obter mais informações sobre a função `mysql_library_init()`.

Usar arquivos de opções pode facilitar a alternância entre uma aplicação cliente/servidor e uma em que o MySQL está embutido. Coloque as opções comuns sob o grupo `[server]`. Essas são lidas por ambas as versões do MySQL. Opções específicas do cliente/servidor devem ir sob a seção `[mysqld]`. Coloque as opções específicas da biblioteca de servidor MySQL embutida na seção `[embedded]`. Opções específicas de aplicativos vão para a seção rotulada `[ApplicationName_SERVER]`. Veja Seção 4.2.2.2, “Usando Arquivos de Opções”.
