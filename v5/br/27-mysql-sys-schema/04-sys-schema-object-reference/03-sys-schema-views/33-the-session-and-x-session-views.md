#### 26.4.3.33 A sessão e as visualizações da sessão x$

Essas visualizações são semelhantes às `processlist` e `x$processlist`, mas elas filtram os processos em segundo plano para exibir apenas as sessões do usuário. Para descrições das colunas, consulte a Seção 26.4.3.22, “As visualizações processlist e x$processlist”.
