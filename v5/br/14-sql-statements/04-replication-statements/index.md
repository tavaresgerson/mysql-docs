[13.4.2 Instruções SQL para Controlar Servidores Replica](replication-statements-replica.html)

A Replication pode ser controlada através da interface SQL usando as instruções descritas nesta seção. As instruções são divididas em um grupo que controla servidores source de replication, um grupo que controla servidores replica e um grupo que pode ser aplicado a quaisquer servidores em uma topology de replication.
