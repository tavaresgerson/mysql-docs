13.4.1 Declarações SQL para controle de servidores de origem de replicação

13.4.2 Declarações SQL para controle de servidores replicados

13.4.3 Declarações SQL para controle da replicação de grupo

A replicação pode ser controlada através da interface SQL usando as instruções descritas nesta seção. As instruções são divididas em um grupo que controla os servidores de origem da replicação, um grupo que controla os servidores de replica e um grupo que pode ser aplicado a qualquer servidor em uma topologia de replicação.
