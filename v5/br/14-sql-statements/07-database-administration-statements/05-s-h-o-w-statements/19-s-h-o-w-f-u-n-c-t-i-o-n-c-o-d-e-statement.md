#### 13.7.5.19 Instrução SHOW FUNCTION CODE

```sql
SHOW FUNCTION CODE func_name
```

Esta instrução é similar a [`SHOW PROCEDURE CODE`](show-procedure-code.html "13.7.5.27 SHOW PROCEDURE CODE Statement"), mas para stored functions. Consulte [Seção 13.7.5.27, “Instrução SHOW PROCEDURE CODE”](show-procedure-code.html "13.7.5.27 SHOW PROCEDURE CODE Statement").
