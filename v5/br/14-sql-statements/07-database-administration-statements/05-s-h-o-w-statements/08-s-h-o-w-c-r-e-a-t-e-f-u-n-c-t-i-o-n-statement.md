#### 13.7.5.8 Instrução SHOW CREATE FUNCTION

```sql
SHOW CREATE FUNCTION func_name
```

Esta instrução é semelhante a [`SHOW CREATE PROCEDURE`](show-create-procedure.html "13.7.5.9 SHOW CREATE PROCEDURE Statement"), mas é para funções armazenadas. Consulte [Seção 13.7.5.9, “Instrução SHOW CREATE PROCEDURE”](show-create-procedure.html "13.7.5.9 SHOW CREATE PROCEDURE Statement").
