### 13.1.13 Instrução CREATE FUNCTION

A instrução `CREATE FUNCTION` é utilizada para criar funções armazenadas (*stored functions*) e funções carregáveis (*loadable functions*):

* Para obter informações sobre a criação de funções armazenadas, consulte Seção 13.1.16, “Instruções CREATE PROCEDURE e CREATE FUNCTION”.

* Para obter informações sobre a criação de funções carregáveis, consulte Seção 13.7.3.1, “Instrução CREATE FUNCTION para Funções Carregáveis”.
