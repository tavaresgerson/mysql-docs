### 13.1.24 DROP FUNCTION Statement

O Statement `DROP FUNCTION` é usado para descartar stored functions e loadable functions:

* Para informações sobre como descartar stored functions, consulte Seção 13.1.27, “DROP PROCEDURE and DROP FUNCTION Statements”.

* Para informações sobre como descartar loadable functions, consulte Seção 13.7.3.2, “DROP FUNCTION Statement for Loadable Functions”.
