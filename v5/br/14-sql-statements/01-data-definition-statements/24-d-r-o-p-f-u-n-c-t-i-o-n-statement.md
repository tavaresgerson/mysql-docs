### 13.1.24 Declaração da função DROP

A instrução `DROP FUNCTION` é usada para descartar funções armazenadas e funções carregáveis:

- Para obter informações sobre a remoção de funções armazenadas, consulte Seção 13.1.27, “Instruções DROP PROCEDURE e DROP FUNCTION”.

- Para obter informações sobre a remoção de funções carregáveis, consulte Seção 13.7.3.2, “Instrução DROP FUNCTION para Funções Carregáveis”.
