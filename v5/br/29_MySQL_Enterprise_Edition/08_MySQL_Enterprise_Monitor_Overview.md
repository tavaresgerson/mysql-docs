## 28.8 Visão geral do Monitoramento Empresarial do MySQL

Para obter informações sobre o comportamento do MySQL Enterprise Monitor, consulte o manual do MySQL Enterprise Monitor: https://dev.mysql.com/doc/mysql-monitor/en/.

Importante

O MySQL Enterprise Monitor será descontinuado e obsoleto a partir de 1º de janeiro de 2025.

Após essa data, o MySQL Enterprise Monitor não receberá mais atualizações de segurança, atualizações não relacionadas à segurança, correções de bugs ou atualizações de conteúdo técnico online. Ele passará para o modelo de Suporte Sustentável.

O que esperar quando o MySQL Enterprise Monitor atingir o fim de vida (EOL):

* O MySQL cessará todas as atividades de correção de bugs para o produto
* O MySQL cessará todas as atividades de correção de segurança para o produto
* O MySQL cessará todas as novas funcionalidades para o produto

O Suporte Sustentado não inclui:

* Novas atualizações de programas, correções, alertas de segurança e atualizações de patches críticos

* Novas atualizações fiscais, legais ou regulatórias
* Novos scripts de atualização
* Certificação com novos produtos/versões de terceiros
* Compromisso e diretrizes de resposta de 24 horas para solicitações de serviço de Grau 1, conforme definido na "Seção 9 - Definições de Graus" no documento intitulado "Políticas de Suporte Técnico de Software Oracle"

* As correções ou atualizações anteriormente lançadas que a Oracle retirou da publicação. Os pacotes de software mais antigos ou existentes publicados permanecerão disponíveis como conteúdo arquivado.

Para o conjunto de Políticas de Suporte Técnico da Oracle, visite: <https://www.oracle.com/support/policies.html>

Para uma explicação dos diferentes modelos de suporte (como Suporte Sustentável), visite: <https://www.oracle.com/support/lifetime-support/>

Para os clientes que estão usando atualmente versões anteriores do MySQL Enterprise Monitor, suas opções incluem:

* Use o Enterprise Manager para MySQL. Este é um produto gratuito para clientes com um contrato de suporte válido da Oracle. Para mais informações, visite [Monitoramento abrangente e Gerenciamento de conformidade para bancos de dados MySQL usando o Enterprise Manager][(https://blogs.oracle.com/observability/post/comprehensive-monitoring-and-compliance-management-for-mysql-databases-using-enterprise-manager)].

* Utilize as capacidades de monitoramento de banco de dados do serviço de Gerenciamento de Banco de Dados OCI e do serviço Ops Insights. Isso está disponível para MySQL HeatWave e MySQL local. Para mais informações, visite [Gerenciamento de Banco de Dados para MySQL HeatWave e MySQL local][(https://blogs.oracle.com/mysql/post/database-management-for-mysql-heatwave)] e [Ops Insights para HeatWave e MySQL local][(https://docs.oracle.com/en-us/iaas/operations-insights/doc/mysql-insights.html)].
