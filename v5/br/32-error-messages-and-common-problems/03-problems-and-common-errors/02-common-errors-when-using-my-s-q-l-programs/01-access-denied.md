#### B.3.2.1 Access denied

Um erro `Access denied` pode ter muitas causas. Frequentemente, o problema está relacionado às contas MySQL que o server permite que os client programs usem ao se conectar. Consulte Seção 6.2, “Controle de Acesso e Gerenciamento de Contas”, e Seção 6.2.17, “Solução de Problemas de Conexão com o MySQL”.
