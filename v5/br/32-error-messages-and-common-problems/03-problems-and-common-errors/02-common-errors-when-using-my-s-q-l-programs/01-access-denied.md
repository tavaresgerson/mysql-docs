#### B.3.2.1 Acesso negado

Um erro de `Acesso negado` pode ter várias causas. Muitas vezes, o problema está relacionado às contas do MySQL que o servidor permite que os programas de cliente usem ao se conectar. Veja [Seção 6.2, “Controle de Acesso e Gerenciamento de Contas”](access-control.html) e [Seção 6.2.17, “Soluções para Problemas de Conexão ao MySQL”](problems-connecting.html).
