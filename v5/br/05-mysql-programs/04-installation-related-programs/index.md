## 4.4 Programas relacionados à instalação

4.4.1 comp\_err — Arquivo de Mensagem de Erro de Compilação do MySQL

4.4.2 mysql\_install\_db — Inicializar o diretório de dados do MySQL

4.4.3 mysql\_plugin — Configurar plugins do servidor MySQL

4.4.4 mysql\_secure\_installation — Melhorar a segurança da instalação do MySQL

4.4.5 mysql\_ssl\_rsa\_setup — Criar arquivos SSL/RSA

4.4.6 mysql\_tzinfo\_to\_sql — Carregar as tabelas de fuso horário

4.4.7 mysql\_upgrade — Verificar e atualizar tabelas do MySQL

Os programas desta seção são usados durante a instalação ou atualização do MySQL.
