## 27.11 APIs Ruby do MySQL

O gem `mysql2` Ruby fornece uma API para conectar-se ao MySQL, realizar consultas e iterar pelos resultados; é destinado a suportar MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página `mysql2` em RubyGems.org (https://rubygems.org/gems/mysql2/) ou a página do projeto no [GitHub][(https://github.com/brianmario/mysql2)].

Para informações sobre o contexto e a sintaxe da linguagem Ruby, consulte a Linguagem de Programação Ruby.

### 27.11.1 API MySQL/Ruby

O módulo MySQL/Ruby fornece acesso a bancos de dados MySQL usando Ruby através de `libmysqlclient`.

Para informações sobre a instalação do módulo e as funções expostas, consulte MySQL/Ruby.

### 27.11. API Ruby/MySQL

O módulo Ruby/MySQL fornece acesso a bancos de dados MySQL usando Ruby através de uma interface de driver nativo que utiliza o protocolo de rede MySQL.

Para informações sobre a instalação do módulo e as funções expostas, consulte Ruby/MySQL.
