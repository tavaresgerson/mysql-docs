O framework de implantação Docker suporta a instalação e configuração fáceis do MySQL Server. Esta seção explica como usar uma imagem do MySQL Server Docker.

Você precisa ter o Docker instalado no seu sistema antes de poder usar uma imagem do Servidor MySQL Docker. Consulte [Instale o Docker](https://docs.docker.com/engine/installation/) para obter instruções.

Aviso

Cuidado com as preocupações de segurança ao executar contêineres Docker. Consulte [Segurança do Docker](https://docs.docker.com/engine/security/) para obter detalhes.
