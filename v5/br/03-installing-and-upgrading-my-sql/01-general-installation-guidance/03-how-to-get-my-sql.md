### 2.1.3 Como obter o MySQL

Consulte nossa página de downloads em <https://dev.mysql.com/downloads/> para obter informações sobre a versão atual do MySQL e instruções de download.

Para plataformas Linux baseadas em RPM que utilizam o Yum como sistema de gerenciamento de pacotes, o MySQL pode ser instalado usando o [Repositório Yum do MySQL](https://dev.mysql.com/downloads/repo/yum/). Consulte a Seção 2.5.1, “Instalando o MySQL no Linux Usando o Repositório Yum do MySQL”, para obter detalhes.

Para plataformas Linux baseadas no Debian, o MySQL pode ser instalado usando o [Repositório MySQL APT](https://dev.mysql.com/downloads/repo/apt/). Consulte a Seção 2.5.3, “Instalando o MySQL no Linux Usando o Repositório MySQL APT”, para obter detalhes.

Para as plataformas SUSE Linux Enterprise Server (SLES), o MySQL pode ser instalado usando o [Repositório MySQL SLES](https://dev.mysql.com/downloads/repo/suse/). Consulte a Seção 2.5.4, “Instalando o MySQL no Linux usando o Repositório MySQL SLES”, para obter detalhes.

Para obter a fonte de desenvolvimento mais recente, consulte a Seção 2.8.5, “Instalando o MySQL usando uma árvore de fonte de desenvolvimento”.
