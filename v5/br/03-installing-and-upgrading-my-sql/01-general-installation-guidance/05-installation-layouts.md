### 2.1.5 Planos de instalação

O layout de instalação difere para diferentes tipos de instalação (por exemplo, pacotes nativos, tarballs binários e tarballs de código-fonte), o que pode levar à confusão ao gerenciar diferentes sistemas ou usar diferentes fontes de instalação. Os layouts individuais são fornecidos no capítulo correspondente ao tipo de instalação ou plataforma, conforme descrito a seguir. Observe que o layout das instalações de fornecedores que não sejam a Oracle pode diferir desses layouts.

- Seção 2.3.1, “Organização da Instalação do MySQL no Microsoft Windows”

- Seção 2.8.3, “Layout do MySQL para Instalação de Fonte”

- Tabela 2.3, “Layout de instalação do MySQL para pacote binário genérico Unix/Linux”

- Tabela 2.12, “Layout de instalação do MySQL para pacotes RPM do Linux da MySQL Developer Zone”

- Tabela 2.7, “Organização da Instalação do MySQL no macOS”
