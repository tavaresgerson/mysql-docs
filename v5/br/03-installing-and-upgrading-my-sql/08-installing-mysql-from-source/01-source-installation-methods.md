### 2.8.1 Métodos de instalação da fonte

Existem dois métodos para instalar o MySQL a partir da fonte:

- Use uma distribuição padrão do MySQL. Para obter uma distribuição padrão, consulte a Seção 2.1.3, “Como obter o MySQL”. Para obter instruções sobre como construir a partir de uma distribuição padrão, consulte a Seção 2.8.4, “Instalando o MySQL usando uma distribuição de fonte padrão”.

  As distribuições padrão estão disponíveis como arquivos **tar** comprimidos, arquivos Zip ou pacotes RPM. Os arquivos de distribuição têm nomes na forma `mysql-VERSION.tar.gz`, `mysql-VERSION.zip` ou `mysql-VERSION.rpm`, onde *`VERSION`* é um número como `5.7.44`. Os nomes dos arquivos de distribuição de fonte podem ser distinguidos daqueles das distribuições binárias pré-compiladas, pois os nomes de distribuição de fonte são genéricos e não incluem o nome da plataforma, enquanto os nomes das distribuições binárias incluem um nome da plataforma que indica o tipo de sistema para o qual a distribuição é destinada (por exemplo, `pc-linux-i686` ou `winx64`).

- Use uma árvore de desenvolvimento do MySQL. Para obter informações sobre a construção a partir de uma das árvores de desenvolvimento, consulte a Seção 2.8.5, “Instalando o MySQL usando uma árvore de fonte de desenvolvimento”.
