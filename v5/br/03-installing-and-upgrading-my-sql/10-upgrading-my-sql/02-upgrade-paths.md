### 2.10.2 Caminhos de atualização

- A atualização só é suportada entre as versões de Disponibilidade Geral (GA).

- A atualização do MySQL 5.6 para 5.7 é suportada. A atualização para a versão mais recente é recomendada antes da atualização para a próxima versão. Por exemplo, faça a atualização para a versão mais recente do MySQL 5.6 antes de atualizar para o MySQL 5.7.

- A atualização que ignora as versões não é suportada. Por exemplo, a atualização direta de MySQL 5.5 para 5.7 não é suportada.

- A atualização dentro de uma série de lançamentos é suportada. Por exemplo, a atualização de MySQL 5.7.*`x`* para 5.7.*`y`* é suportada. A omissão de um lançamento também é suportada. Por exemplo, a atualização de MySQL 5.7.*`x`* para 5.7.*`z`* é suportada.
