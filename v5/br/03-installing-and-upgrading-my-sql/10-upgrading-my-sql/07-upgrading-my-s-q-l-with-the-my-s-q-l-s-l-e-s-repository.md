### 2.10.7 Atualização do MySQL com o Repositório MySQL SLES

Na plataforma SUSE Linux Enterprise Server (SLES), para realizar uma atualização local do MySQL e seus componentes, use o repositório SLES do MySQL. Veja Atualizando o MySQL com o repositório SLES do MySQL em Um guia rápido para usar o repositório SLES do MySQL.
