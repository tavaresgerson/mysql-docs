### 19.2.1 Instalando o MySQL Shell

19.2.1.1 Instalando o MySQL Shell no Microsoft Windows

19.2.1.2 Instalando o MySQL Shell no Linux

19.2.1.3 Instalando o MySQL Shell no macOS

Esta seção descreve como baixar, instalar e iniciar o MySQL Shell, que é uma interface interativa (JavaScript, Python ou SQL) que oferece suporte ao desenvolvimento e à administração do MySQL Server. O MySQL Shell é um componente que você pode instalar separadamente.

#### Requisitos

O MySQL Shell está disponível para Microsoft Windows, Linux e macOS para plataformas de 64 bits. O MySQL Shell requer que o X Plugin integrado esteja ativo. Você pode instalar o plugin do Server antes ou depois de instalar o MySQL Shell. Para obter instruções, consulte Instalando o X Plugin.
