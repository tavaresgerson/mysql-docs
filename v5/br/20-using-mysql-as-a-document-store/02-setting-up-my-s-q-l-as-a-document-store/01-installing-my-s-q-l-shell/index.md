### 19.2.1 Instalar o MySQL Shell

19.2.1.1 Instalando o MySQL Shell no Microsoft Windows

19.2.1.2 Instalar o MySQL Shell no Linux

19.2.1.3 Instalando o MySQL Shell no macOS

Esta seção descreve como fazer o download, instalar e iniciar o MySQL Shell, que é uma interface interativa de JavaScript, Python ou SQL que suporta o desenvolvimento e a administração do MySQL Server. O MySQL Shell é um componente que você pode instalar separadamente.

#### Requisitos

O MySQL Shell está disponível no Microsoft Windows, Linux e macOS para plataformas de 64 bits. O MySQL Shell exige que o Plugin X integrado esteja ativo. Você pode instalar o plugin do servidor antes ou depois de instalar o MySQL Shell. Para obter instruções, consulte Instalar o Plugin X.
