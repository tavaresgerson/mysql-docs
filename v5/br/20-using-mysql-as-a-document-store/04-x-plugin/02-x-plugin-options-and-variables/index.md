19.4.2.2 Opções e Variáveis de Sistema do X Plugin

19.4.2.3 Variáveis de Status do X Plugin

Esta seção descreve as opções de comando e as variáveis de sistema que configuram o X Plugin, bem como as variáveis de status disponíveis para fins de monitoramento. Se os valores de configuração especificados no momento da inicialização (startup time) estiverem incorretos, o X Plugin poderá falhar ao inicializar corretamente e o Server não o carregará. Neste caso, o Server também poderá gerar mensagens de erro para outras configurações do X Plugin porque não consegue reconhecê-las.
