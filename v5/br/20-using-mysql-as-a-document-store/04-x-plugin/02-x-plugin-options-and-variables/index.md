19.4.2.3 Variáveis de Status do Plugin

Esta seção descreve as opções de comando e as variáveis de sistema que configuram o X Plugin, bem como as variáveis de status disponíveis para fins de monitoramento. Se os valores de configuração especificados no momento do início forem incorretos, o X Plugin pode não ser iniciado corretamente e o servidor não o carregará. Nesse caso, o servidor também pode produzir mensagens de erro para outras configurações do X Plugin, pois não consegue reconhecê-las.
