### 2.1.3 Como Obter o MySQL

Verifique nossa página de downloads em <https://dev.mysql.com/downloads/> para obter informações sobre a versão atual do MySQL e instruções de download.

Para plataformas Linux baseadas em RPM que usam Yum como sistema de gerenciamento de pacotes, o MySQL pode ser instalado usando o [MySQL Yum Repository](https://dev.mysql.com/downloads/repo/yum/). Consulte a Seção 2.5.1, "Instalando o MySQL no Linux Usando o MySQL Yum Repository", para detalhes.

Para plataformas Linux baseadas em Debian, o MySQL pode ser instalado usando o [MySQL APT Repository](https://dev.mysql.com/downloads/repo/apt/). Consulte a Seção 2.5.3, "Instalando o MySQL no Linux Usando o MySQL APT Repository", para detalhes.

Para plataformas SUSE Linux Enterprise Server (SLES), o MySQL pode ser instalado usando o [MySQL SLES Repository](https://dev.mysql.com/downloads/repo/suse/). Consulte a Seção 2.5.4, "Instalando o MySQL no Linux Usando o MySQL SLES Repository", para detalhes.

Para obter o código-fonte de desenvolvimento mais recente, consulte a Seção 2.8.5, "Instalando o MySQL Usando uma Árvore de Código-Fonte de Desenvolvimento".
