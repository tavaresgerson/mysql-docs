### 2.8.3 Layout do MySQL para Source Installation

Por padrão, quando você instala o MySQL após compilá-lo a partir do código-fonte, a etapa de instalação coloca os arquivos em `/usr/local/mysql`. A localização dos componentes no diretório de instalação é a mesma que para distribuições binárias. Consulte Tabela 2.3, “Layout de Instalação do MySQL para Pacotes Binários Genéricos Unix/Linux”, e Seção 2.3.1, “Layout de Instalação do MySQL no Microsoft Windows”. Para configurar locais de instalação diferentes dos padrões, utilize as opções descritas na Seção 2.8.7, “Opções de Source-Configuration do MySQL”.
