Para uma lista das versões do macOS suportadas pelo servidor MySQL, consulte <https://www.mysql.com/support/supportedplatforms/database.html>.

O MySQL para macOS está disponível em várias formas diferentes:

* Instalador de Pacote Nativo (Native Package Installer), que usa o instalador nativo do macOS (DMG) para guiar você pela instalação do MySQL. Para mais informações, consulte a Seção 2.4.2, “Instalação do MySQL no macOS Usando Pacotes Nativos”. Você pode usar o instalador de pacote no macOS. O usuário que você usar para realizar a instalação deve ter privilégios de administrador.

* Arquivo TAR Compactado (Compressed TAR archive), que usa um arquivo empacotado usando os comandos Unix **tar** e **gzip**. Para usar este método, você precisa abrir uma janela do **Terminal**. Você não precisa de privilégios de administrador usando este método, pois pode instalar o servidor MySQL em qualquer lugar. Para mais informações sobre como usar este método, você pode seguir as instruções genéricas para usar um tarball, Seção 2.2, “Instalação do MySQL em Unix/Linux Usando Binários Genéricos”.

  Além da instalação principal, o Instalador de Pacote também inclui a Seção 2.4.3, “Instalação de um Daemon de Lançamento do MySQL” e a Seção 2.4.4, “Instalação e Uso do Painel de Preferências do MySQL”, ambas simplificando o gerenciamento da sua instalação.

Para informações adicionais sobre o uso do MySQL no macOS, consulte a Seção 2.4.1, “Notas Gerais sobre a Instalação do MySQL no macOS”.
