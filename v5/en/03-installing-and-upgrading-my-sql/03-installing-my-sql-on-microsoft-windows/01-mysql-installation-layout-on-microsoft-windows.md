### 2.3.1 Estrutura da Instalação do MySQL no Microsoft Windows

Para o MySQL 5.7 no Windows, o diretório de instalação padrão é `C:\Program Files\MySQL\MySQL Server 5.7` para instalações realizadas com o MySQL Installer. Se você usar o método de arquivo ZIP para instalar o MySQL, você pode preferir instalar em `C:\mysql`. No entanto, a estrutura dos subdiretórios permanece a mesma.

Todos os arquivos estão localizados dentro deste diretório pai, usando a estrutura mostrada na tabela a seguir.
