### 2.10.2 Caminhos de Upgrade

* O Upgrade é suportado apenas entre General Availability (GA) releases.

* O Upgrade do MySQL 5.6 para 5.7 é suportado. É recomendável fazer o upgrade para a última release antes de fazer o upgrade para a próxima version. Por exemplo, faça o upgrade para a última MySQL 5.6 release antes de fazer o upgrade para o MySQL 5.7.

* O Upgrade que pula versions não é suportado. Por exemplo, fazer o upgrade diretamente do MySQL 5.5 para 5.7 não é suportado.

* O Upgrade dentro de uma release series é suportado. Por exemplo, fazer o upgrade do MySQL 5.7.*`x`* para 5.7.*`y`* é suportado. Pular uma release também é suportado. Por exemplo, fazer o upgrade do MySQL 5.7.*`x`* para 5.7.*`z`* é suportado.
