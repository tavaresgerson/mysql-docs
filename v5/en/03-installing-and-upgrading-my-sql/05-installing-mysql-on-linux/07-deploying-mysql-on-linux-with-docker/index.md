A estrutura de *deployment* do Docker suporta a fácil *installation* e *configuration* do MySQL Server. Esta seção explica como usar uma *Image* Docker do MySQL Server.

Você precisa ter o Docker instalado em seu sistema antes de poder usar uma *Image* Docker do MySQL Server. Consulte [Install Docker](https://docs.docker.com/engine/installation/) para obter as instruções.

Aviso

Atente-se às preocupações de *security* ao executar *Containers* Docker. Consulte [Docker security](https://docs.docker.com/engine/security/) para obter detalhes.

As instruções para usar o *Container* Docker do MySQL estão divididas em duas seções.
