### 8.3.6 Verifying Index Usage

Always check whether all your queries really use the indexes
that you have created in the tables. Use the
[`EXPLAIN`](explain.html "13.8.2 EXPLAIN Statement") statement, as described
in [Section 8.8.1, “Optimizing Queries with EXPLAIN”](using-explain.html "8.8.1 Optimizing Queries with EXPLAIN").
