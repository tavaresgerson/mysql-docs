### 23.3.2 Trigger Metadata

To obtain metadata about triggers:

* Query the [`TRIGGERS`](information-schema-triggers-table.html "24.3.29 The INFORMATION_SCHEMA TRIGGERS Table") table of the
  `INFORMATION_SCHEMA` database. See
  [Section 24.3.29, “The INFORMATION\_SCHEMA TRIGGERS Table”](information-schema-triggers-table.html "24.3.29 The INFORMATION_SCHEMA TRIGGERS Table").

* Use the [`SHOW CREATE TRIGGER`](show-create-trigger.html "13.7.5.11 SHOW CREATE TRIGGER Statement")
  statement. See [Section 13.7.5.11, “SHOW CREATE TRIGGER Statement”](show-create-trigger.html "13.7.5.11 SHOW CREATE TRIGGER Statement").

* Use the [`SHOW TRIGGERS`](show-triggers.html "13.7.5.38 SHOW TRIGGERS Statement")
  statement. See [Section 13.7.5.38, “SHOW TRIGGERS Statement”](show-triggers.html "13.7.5.38 SHOW TRIGGERS Statement").
