#### 5.4.2.4 Filtragem do Error Log

A `system variable` [`log_error_verbosity`](server-system-variables.html#sysvar_log_error_verbosity) controla a verbosidade do `server` para a escrita de mensagens de error, warning e note no `error log`. Os valores permitidos são 1 (apenas errors), 2 (errors e warnings), 3 (errors, warnings e notes), sendo 3 o valor padrão. Se o valor for maior que 2, o `server` registra `aborted connections` e `access-denied errors` para novas tentativas de conexão. Consulte [Seção B.3.2.9, “Communication Errors and Aborted Connections”](communication-errors.html "B.3.2.9 Communication Errors and Aborted Connections").
