#### 5.1.12.2 Configurando o MySQL Server para Permitir Conexões IPv6

O MySQL server escuta em um único *network socket* para conexões TCP/IP. Este *socket* é ligado a um único endereço, mas é possível que um endereço mapeie para múltiplas interfaces de rede. Para especificar um endereço, defina [`bind_address=addr`](server-system-variables.html#sysvar_bind_address) na inicialização do server, onde *`addr`* é um endereço IPv4 ou IPv6 ou um *host name*. Para detalhes, consulte a descrição de [`bind_address`](server-system-variables.html#sysvar_bind_address) na [Seção 5.1.7, “Server System Variables”](server-system-variables.html "5.1.7 Server System Variables").
