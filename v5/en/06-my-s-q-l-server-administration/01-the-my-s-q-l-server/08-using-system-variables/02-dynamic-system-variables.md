#### 5.1.8.2 Variáveis de Sistema Dinâmicas

Muitas variáveis de sistema do servidor são dinâmicas e podem ser configuradas em tempo de execução (`runtime`). Consulte [Seção 13.7.4.1, “Sintaxe SET para Atribuição de Variáveis”](set-variable.html "13.7.4.1 Sintaxe SET para Atribuição de Variáveis"). Para uma descrição dos requisitos de privilégio para definir variáveis de sistema, consulte [Seção 5.1.8.1, “Privilégios de Variáveis de Sistema”](system-variable-privileges.html "5.1.8.1 Privilégios de Variáveis de Sistema")

A tabela a seguir lista todas as variáveis de sistema dinâmicas aplicáveis dentro do `mysqld`.

A tabela lista o tipo de dado e o escopo de cada variável. A última coluna indica se o escopo de cada variável é Global, Session, ou ambos. Consulte as descrições dos itens correspondentes para obter detalhes sobre como configurar e usar as variáveis. Quando apropriado, links diretos para informações adicionais sobre os itens são fornecidos.

Variáveis do tipo "string" aceitam um valor string. Variáveis do tipo "numeric" aceitam um valor numérico. Variáveis do tipo "boolean" podem ser definidas como 0, 1, `ON` ou `OFF`. Variáveis marcadas como "enumeration" (enumeração) normalmente devem ser definidas para um dos valores disponíveis para a variável, mas também podem ser definidas para o número que corresponde ao valor de enumeração desejado. Para variáveis de sistema enumeradas, o primeiro valor de enumeração corresponde a 0. Isso difere do tipo de dado [`ENUM`](enum.html "11.3.5 O Tipo ENUM") usado para colunas de tabela, para o qual o primeiro valor de enumeração corresponde a 1.
