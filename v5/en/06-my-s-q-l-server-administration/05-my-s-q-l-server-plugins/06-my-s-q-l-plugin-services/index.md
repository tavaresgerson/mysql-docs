MySQL server plugins have access to server “plugin services.” The plugin services interface complements the plugin API by exposing server functionality that plugins can call. For developer information about writing plugin services, see [MySQL Services for Plugins](/doc/extending-mysql/5.7/en/services-for-plugins.html). The following sections describe plugin services available at the SQL and C-language levels.
