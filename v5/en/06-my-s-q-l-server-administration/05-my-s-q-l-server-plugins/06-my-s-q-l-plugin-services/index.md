Plugins do server MySQL têm acesso aos “serviços de plugin” do server. A interface de serviços de plugin complementa a Plugin API, expondo funcionalidades do server que os plugins podem chamar. Para informações para desenvolvedores sobre a escrita de serviços de plugin, consulte [MySQL Services for Plugins](/doc/extending-mysql/5.7/en/services-for-plugins.html). As seções a seguir descrevem os serviços de plugin disponíveis nos níveis SQL e C-language.
