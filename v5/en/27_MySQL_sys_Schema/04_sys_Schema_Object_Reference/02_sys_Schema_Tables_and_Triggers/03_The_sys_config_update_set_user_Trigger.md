#### 26.4.2.3 The sys\_config\_update\_set\_user Trigger

The [`sys_config_update_set_user`](sys-sys-config-update-set-user.html "26.4.2.3 The sys_config_update_set_user Trigger")
trigger for the [`sys_config`](sys-sys-config.html "26.4.2.1 The sys_config Table") table
is similar to the
[`sys_config_insert_set_user`](sys-sys-config-insert-set-user.html "26.4.2.2 The sys_config_insert_set_user Trigger")
trigger, but for [`UPDATE`](update.html "13.2.11 UPDATE Statement")
statements.
