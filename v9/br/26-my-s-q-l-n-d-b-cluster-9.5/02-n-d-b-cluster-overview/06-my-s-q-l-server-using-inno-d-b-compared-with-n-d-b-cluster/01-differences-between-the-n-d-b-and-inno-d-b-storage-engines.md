#### 25.2.6.1 Diferenças entre os motores de armazenamento NDB e InnoDB

O motor de armazenamento `NDB` é implementado usando uma arquitetura distribuída, sem compartilhamento, o que faz com que ele se comporte de maneira diferente do `InnoDB` de várias maneiras. Para aqueles não acostumados a trabalhar com `NDB`, comportamentos inesperados podem surgir devido à sua natureza distribuída em relação a transações, chaves estrangeiras, limites de tabela e outras características.
