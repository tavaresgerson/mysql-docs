#### 25.6.2.2 Mensagens de inicialização do log do cluster NDB

As possíveis mensagens de inicialização com descrições estão listadas na seguinte tabela:

* `Início inicial, aguardando que %s se conecte, nós [ todos: %s conectados: %s sem espera: %s ]`

* `Aguardando que os nós: %s se conectem, nós [ todos: %s conectados: %s sem espera: %s ]`

* `Aguardando %u segundos para que os nós %s se conectem, nós [ todos: %s conectados: %s sem espera: %s ]`

* `Aguardando início não particionado, nós [ todos: %s conectados: %s ausentes: %s sem espera: %s ]`

* `Aguardando %u segundos para o início não particionado, nós [ todos: %s conectados: %s ausentes: %s sem espera: %s ]`

* `Início inicial com nós %s [ ausentes: %s sem espera: %s ]`

* `Início com todos os nós %s`

* `Início com nós %s [ ausentes: %s sem espera: %s ]`

* `Início potencialmente particionado com nós %s [ ausentes: %s sem espera: %s ]`

* `Relatório de início desconhecido: 0x%x [ %s %s %s %s ]`
