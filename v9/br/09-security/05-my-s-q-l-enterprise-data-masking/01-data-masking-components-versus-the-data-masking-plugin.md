### 8.5.1 Componentes de Máscara de Dados vs. Plugin de Máscara de Dados

Anteriormente, o MySQL habilitava as capacidades de mascaramento e desidentificação usando um plugin do lado do servidor, mas passou a usar a infraestrutura de componentes como uma implementação alternativa. A tabela a seguir compara brevemente os componentes de Máscara de Dados do MySQL Enterprise e a biblioteca de plugins para fornecer uma visão geral de suas diferenças. Isso pode ajudá-lo a fazer a transição do plugin para os componentes.

Nota

Só os componentes de Máscara de Dados ou o plugin devem ser habilitados de cada vez. Habilitar ambos os componentes e o plugin não é suportado e os resultados podem não ser os esperados.
