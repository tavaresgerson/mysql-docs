10.4.3.2 Desvantagens de Criar Muitas Tabelas na Mesma Base de Dados

Algumas técnicas para manter consultas individuais rápidas envolvem a divisão dos dados em várias tabelas. Quando o número de tabelas chega a milhares ou até milhões, o custo de lidar com todas essas tabelas se torna uma nova consideração de desempenho.
