O componente Rastreador de Opções fornece informações sobre as funcionalidades do servidor MySQL, bem como sobre os componentes e plugins do MySQL instalados no sistema:

* Finalidade: Fornecer informações sobre as funcionalidades do MySQL. Essas funcionalidades podem ser integradas ao próprio servidor MySQL ou implementadas por componentes ou plugins instalados no sistema.

* URN: `file://component_option_tracker`

Observação

Neste contexto, referimo-nos a essas funcionalidades como “opções”. Não devem ser confundidas com as opções de linha de comando usadas com programas do MySQL, como o **mysqld** ou o cliente **mysql**.

Este componente está disponível como parte da Edição Empresarial do MySQL, uma oferta comercial.

Consulte a Seção 7.5.8.2, “Componentes Compatíveis com Rastreador de Opções”, para obter informações sobre as funcionalidades e componentes que suportam o Rastreador de Opções.
