### 7.1.4 Referência de Opções do Servidor, Variáveis de Sistema e Variáveis de Status

A tabela a seguir lista todas as opções de linha de comando, variáveis de sistema e variáveis de status aplicáveis no `mysqld`.

A tabela lista opções de linha de comando (Cmd-line), opções válidas em arquivos de configuração (Arquivo de opção), variáveis de sistema do servidor (System Var) e variáveis de status (Status var) em uma lista unificada, com uma indicação de onde cada opção ou variável é válida. Se um conjunto de opções do servidor definido na linha de comando ou em um arquivo de opção difere do nome da variável de sistema correspondente, o nome da variável é anotado imediatamente abaixo da opção correspondente. Para variáveis de sistema e status, o escopo da variável (Var Scope) é Global, Sessão ou ambos. Consulte as descrições dos itens correspondentes para obter detalhes sobre a configuração e uso das opções e variáveis. Quando apropriado, são fornecidos links diretos para informações adicionais sobre os itens.
