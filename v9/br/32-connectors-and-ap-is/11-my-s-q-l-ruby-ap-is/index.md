31.11.1 API MySQL/Ruby

31.11.2 API Ruby/MySQL

O gem `mysql2` Ruby fornece uma API para conectar-se ao MySQL, realizar consultas e iterar por resultados; ele é destinado a suportar MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página [`mysql2` no RubyGems.org](https://rubygems.org/gems/mysql2/) ou a página do projeto no [GitHub](https://github.com/brianmario/mysql2).

Para informações de contexto e sintaxe sobre a linguagem Ruby, consulte a Linguagem de Programação Ruby.
