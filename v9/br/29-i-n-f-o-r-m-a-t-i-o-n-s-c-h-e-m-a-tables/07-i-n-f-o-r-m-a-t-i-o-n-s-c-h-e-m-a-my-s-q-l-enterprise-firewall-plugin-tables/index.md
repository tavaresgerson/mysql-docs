28.7.2 A Tabela do MySQL Schema Firewall\_USERS

28.7.3 A Tabela do MySQL Schema Firewall\_WHITELIST

As seções a seguir descrevem as tabelas do `MySQL Schema` associadas ao plugin do Firewall do MySQL Enterprise (consulte a Seção 8.4.8.1, “O Plugin do Firewall do MySQL Enterprise”). Elas fornecem visões sobre o cache de dados em memória do firewall. Essas tabelas estão disponíveis apenas se os plugins de firewall apropriados estiverem habilitados. Como o plugin de firewall, essas tabelas são desatualizadas e sujeitas à remoção em uma versão futura do MySQL.

Essas tabelas do `MySQL Schema` *não* são usadas ou suportadas pelo componente do Firewall do MySQL Enterprise.
