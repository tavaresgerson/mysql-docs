28.4.29 A Tabela `INFORMATION_SCHEMA INNODB_VIRTUAL`

Esta seção fornece definições de tabelas para as tabelas `INFORMATION_SCHEMA` `InnoDB`. Para informações relacionadas e exemplos, consulte a Seção 17.15, “Tabelas `INFORMATION_SCHEMA` de InnoDB”.

As tabelas `INFORMATION_SCHEMA` `InnoDB` podem ser usadas para monitorar a atividade contínua do `InnoDB`, para detectar ineficiências antes que se tornem problemas ou para solucionar problemas de desempenho e capacidade. À medida que seu banco de dados se torna maior e mais ocupado, enfrentando os limites da capacidade do hardware, você monitora e ajusta esses aspectos para manter o banco de dados funcionando de forma suave.
