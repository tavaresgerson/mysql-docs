#### 2.3.3.1 Extrair o Arquivo de Instalação

Para instalar o MySQL manualmente, faça o seguinte:

1. Se você estiver atualizando de uma versão anterior, consulte a Seção 3.11, “Atualizando o MySQL no Windows”, antes de iniciar o processo de atualização.

2. Certifique-se de estar logado como um usuário com privilégios de administrador.

3. Escolha um local de instalação. Tradicionalmente, o servidor MySQL é instalado em `C:\mysql`. Se você não instalar o MySQL em `C:\mysql`, você deve especificar o caminho para o diretório de instalação durante a inicialização ou em um arquivo de opção. Consulte a Seção 2.3.3.2, “Criando um Arquivo de Opção”.

   Observação

   O MSI instala o MySQL em `C:\Program Files\MySQL\MySQL Server 9.5`.

4. Extraia o arquivo de instalação para o local de instalação escolhido usando sua ferramenta de compactação de arquivos preferida. Algumas ferramentas podem extrair o arquivo para uma pasta dentro do local de instalação escolhido. Se isso ocorrer, você pode mover o conteúdo da subpasta para o local de instalação escolhido.
