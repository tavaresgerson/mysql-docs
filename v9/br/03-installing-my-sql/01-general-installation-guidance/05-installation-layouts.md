### 2.1.5 Planos de Instalação

O plano de instalação difere para diferentes tipos de instalação (por exemplo, pacotes nativos, tarballs binários e tarballs de código-fonte), o que pode levar à confusão ao gerenciar diferentes sistemas ou usar diferentes fontes de instalação. Os planos individuais estão descritos no capítulo correspondente ao tipo de instalação ou plataforma, conforme descrito a seguir. Observe que o plano de instalações de fornecedores diferentes da Oracle pode diferir desses planos.

* Plano de Instalação do MySQL no Microsoft Windows
* Seção 2.8.3, “Plano de MySQL para Instalação de Código-Fonte”
* Tabela 2.3, “Plano de Instalação do MySQL para Pacotes Binários Genéricos Unix/Linux”
* Tabela 2.13, “Plano de Instalação do MySQL para Pacotes RPM do Linux da MySQL Developer Zone”
* Tabela 2.8, “Plano de Instalação do MySQL no macOS”
