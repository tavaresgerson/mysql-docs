Para uma lista das versões do macOS suportadas pelo servidor MySQL, consulte <https://www.mysql.com/support/supportedplatforms/database.html>.

O MySQL para macOS está disponível em várias formas diferentes:

* Instalador de Pacotes Nativo, que usa o instalador nativo do macOS (DMG) para guiá-lo pela instalação do MySQL. Para mais informações, consulte a Seção 2.4.2, “Instalando o MySQL no macOS Usando Pacotes Nativos”. Você pode usar o instalador de pacotes com o macOS. O usuário que você usa para realizar a instalação deve ter privilégios de administrador.

* Arquivo TAR comprimido, que usa um arquivo compactado usando os comandos Unix **tar** e **gzip**. Para usar esse método, você precisa abrir uma janela do **Terminal**. Você não precisa de privilégios de administrador usando esse método; você pode instalar o servidor MySQL em qualquer lugar usando esse método. Para mais informações sobre como usar esse método, você pode usar as instruções genéricas para usar um tarball, Seção 2.2, “Instalando o MySQL no Unix/Linux Usando Binários Genéricos”.

Além da instalação principal, o Instalador de Pacotes também inclui a Seção 2.4.3, “Instalando e Usando o Daemon de Inicialização do MySQL” e a Seção 2.4.4, “Instalando e Usando o Painel de Preferências do MySQL” para simplificar a gestão da sua instalação.

Para informações adicionais sobre como usar o MySQL no macOS, consulte a Seção 2.4.1, “Notas Gerais sobre a Instalação do MySQL no macOS”.
