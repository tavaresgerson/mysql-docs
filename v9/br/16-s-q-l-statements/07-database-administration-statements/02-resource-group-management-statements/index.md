### 15.7.2 Declarações de Gerenciamento de Grupos de Recursos

15.7.2.1 Declaração ALTER RESOURCE GROUP

15.7.2.2 Declaração CREATE RESOURCE GROUP

15.7.2.3 Declaração DROP RESOURCE GROUP

15.7.2.4 Declaração SET RESOURCE GROUP

O MySQL suporta a criação e o gerenciamento de grupos de recursos e permite a atribuição de threads que estão em execução no servidor a grupos específicos, para que as threads sejam executadas de acordo com os recursos disponíveis para o grupo. Esta seção descreve as declarações SQL disponíveis para o gerenciamento de grupos de recursos. Para uma discussão geral sobre a capacidade de grupos de recursos, consulte a Seção 7.1.16, “Grupos de Recursos”.
