O MySQL fornece funções para realizar várias operações em dados espaciais. Essas funções podem ser agrupadas em várias categorias principais de acordo com o tipo de operação que realizam:

* Funções que criam geometrias em vários formatos (WKT, WKB, interno)

* Funções que convertem geometrias entre formatos
* Funções que acessam propriedades qualitativas ou quantitativas de uma geometria

* Funções que descrevem relações entre duas geometrias
* Funções que criam novas geometrias a partir de geometrias existentes

Para informações gerais sobre o suporte do MySQL para o uso de dados espaciais, consulte a Seção 13.4, “Tipos de Dados Espaciais”.
