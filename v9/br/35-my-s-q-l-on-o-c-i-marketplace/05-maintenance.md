## 34.5 Manutenção

Este produto é gerenciado pelo usuário, o que significa que você é responsável pelas atualizações e manutenção.

### Atualização do MySQL

A instalação existente é baseada em RPM. Para atualizar o servidor MySQL, consulte a Seção 3.7, “Atualizando Instalações com Binários ou Pacotes RPM no Unix/Linux”.

Você pode usar o `scp` para copiar o RPM necessário para a Instância de Computação do OCI ou copiá-lo do Armazenamento de Objetos do OCI, se você configurou o acesso a ele. O Armazenamento de Arquivos também é uma opção. Para mais informações, consulte [Armazenamento de Arquivos e NFS](https://docs.cloud.oracle.com/iaas/Content/File/Concepts/filestorageoverview.htm).

### Backup e Restauração

O Backup do MySQL Enterprise é a solução de backup e restauração preferida. Para mais informações, consulte Backup para Armazenamento em Nuvem.

Para informações sobre o Backup do MySQL Enterprise, consulte Começando com o Backup do MySQL Enterprise.

Para informações sobre o backup e restauração padrão do MySQL, consulte o Capítulo 9, *Backup e Recuperação*.
