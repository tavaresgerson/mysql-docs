Este capítulo descreve como implantar a Edição Empresarial do MySQL como uma Aplicação do Mercado Oracle Cloud Infrastructure. Este é um produto BYOL.

Nota

Para obter mais informações sobre o Mercado Oracle Cloud, consulte [Visão geral do Mercado](https://docs.cloud.oracle.com/iaas/Content/Marketplace/Concepts/marketoverview.htm).

A Aplicação do Mercado da Edição Empresarial do MySQL é uma instância de computação do OCI, executando o Oracle Linux 9.3, com o MySQL EE 8.4. A instalação do MySQL EE na imagem implantada é semelhante à instalação RPM, conforme descrito na Seção 2.5.4, "Instalando o MySQL no Linux usando pacotes RPM da Oracle".

Para obter mais informações sobre a Edição Empresarial do MySQL, consulte o Capítulo 32, *Edição Empresarial do MySQL*.

Para obter mais informações sobre a configuração avançada do MySQL, consulte o Guia de Implantação Segura.

Para obter mais informações sobre o Oracle Linux 9, consulte [Documentação do Oracle Linux](https://docs.oracle.com/en/operating-systems/oracle-linux/9/)
