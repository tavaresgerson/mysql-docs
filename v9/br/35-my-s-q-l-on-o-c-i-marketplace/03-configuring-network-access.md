## 34.3 Configurando Acesso à Rede

Para obter informações sobre as Regras de Segurança do OCI, consulte [Regras de Segurança](https://docs.cloud.oracle.com/iaas/Content/Network/Concepts/securityrules.htm).

Importante

Você deve habilitar o ingress nas seguintes portas:

* 22: SSH
* 3306: MySQL
* 33060: (opcional) Protocolo MySQL X. Usado pelo MySQL Shell.
