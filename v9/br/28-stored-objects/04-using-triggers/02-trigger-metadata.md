### 27.4.2 Metadados dos gatilhos

Para obter metadados sobre os gatilhos:

* Faça uma consulta à tabela `TRIGGERS` do banco de dados `INFORMATION_SCHEMA`. Consulte a Seção 28.3.50, “A tabela TRIGGERS do INFORMATION_SCHEMA”.

* Use a instrução `SHOW CREATE TRIGGER`. Consulte a Seção 15.7.7.13, “Instrução SHOW CREATE TRIGGER”.

* Use a instrução `SHOW TRIGGERS`. Consulte a Seção 15.7.7.41, “Instrução SHOW TRIGGERS”.
