### 27.6.5 Visualizar metadados

Para obter metadados sobre visualizações:

* Faça uma consulta à tabela `VIEWS` do banco de dados `INFORMATION_SCHEMA`. Consulte a Seção 28.3.53, “A tabela VIEWS do INFORMATION_SCHEMA”.

* Use a instrução `SHOW CREATE VIEW`. Consulte a Seção 15.7.7.15, “Instrução SHOW CREATE VIEW”.
