O MySQL suporta visualizações, incluindo visualizações atualizáveis. As visualizações são consultas armazenadas que, quando invocadas, produzem um conjunto de resultados. Uma visualização age como uma tabela virtual.

A discussão a seguir descreve a sintaxe para criar e descartar visualizações, e mostra alguns exemplos de como usá-las.
