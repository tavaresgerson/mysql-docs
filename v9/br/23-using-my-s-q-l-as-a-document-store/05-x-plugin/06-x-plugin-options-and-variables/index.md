### 22.5.6 Opções e variáveis do plugin X

22.5.6.1 Referência de opções e variáveis do plugin X

22.5.6.2 Opções e variáveis do sistema do plugin X

22.5.6.3 Variáveis de status do plugin X

Esta seção descreve as opções de comando e variáveis do sistema que configuram o plugin X, bem como as variáveis de status disponíveis para monitoramento. Se os valores de configuração especificados no momento do início forem incorretos, o plugin X pode não se inicializar corretamente e o servidor não o carregará. Nesse caso, o servidor também pode produzir mensagens de erro para outras configurações do plugin X, pois não consegue reconhecê-las.
