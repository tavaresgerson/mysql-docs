## 22.1 Interfaces para uma Armazenamento de Documentos MySQL

Para trabalhar com o MySQL como um armazenamento de documentos, você usa componentes dedicados e uma escolha de clientes que permitem a comunicação com o servidor MySQL para desenvolver aplicações baseadas em documentos.

* Os seguintes produtos MySQL suportam o Protocolo X e permitem que você use o X DevAPI em sua língua escolhida para desenvolver aplicações que se comunicam com um servidor MySQL funcionando como um armazenamento de documentos:

  + MySQL Shell (que fornece implementações do X DevAPI em JavaScript e Python)

  + Connector/C++
  + Connector/J
  + Connector/Node.js
  + Connector/NET
  + Connector/Python
* O MySQL Shell é uma interface interativa com o MySQL que suporta modos JavaScript, Python ou SQL. Você pode usar o MySQL Shell para prototipar aplicações, executar consultas e atualizar dados. A instalação do MySQL Shell tem instruções para baixar e instalar o MySQL Shell.

* As guias de início rápido (tutoriais) neste capítulo ajudam você a começar a usar o MySQL Shell com o MySQL como um armazenamento de documentos.

  A guia de início rápido para JavaScript está aqui: Seção 22.3, “Guia de Início Rápido em JavaScript: MySQL Shell para Armazenamento de Documentos”.

  A guia de início rápido para Python está aqui: Seção 22.4, “Guia de Início Rápido em Python: MySQL Shell para Armazenamento de Documentos”.

* O *Guia do Usuário do MySQL Shell* no MySQL Shell 9.5 fornece informações detalhadas sobre a configuração e o uso do MySQL Shell.
