Este guia rápido fornece instruções para começar a prototipar aplicativos de armazenamento de documentos interativamente com o Shell MySQL. O guia inclui os seguintes tópicos:

* Introdução à funcionalidade do MySQL, ao Shell MySQL e ao esquema de exemplo `world_x`.

* Operações para gerenciar coleções e documentos.
* Operações para gerenciar tabelas relacionais.
* Operações que se aplicam a documentos dentro das tabelas.

Para seguir este guia rápido, você precisa de um servidor MySQL com o Plugin X instalado, o padrão no 9.5, e o Shell MySQL para usar como cliente. O Shell MySQL 9.5 fornece informações mais detalhadas sobre o Shell MySQL. O Armazenamento de Documentos é acessado usando o X DevAPI, e o Shell MySQL fornece essa API tanto em JavaScript quanto em Python.
