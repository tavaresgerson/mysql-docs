#### 30.4.4.15 O procedimento ps_setup_show_disabled()

Exibe todas as configurações da Schema de Desempenho atualmente desativadas.

##### Parâmetros

* `in_show_instruments BOOLEAN`: Se os instrumentos desativados devem ser exibidos. Isso pode ser uma longa lista.

* `in_show_threads BOOLEAN`: Se as threads desativadas devem ser exibidas.
