#### 30.4.3.20 As visualizações \_global\_total\_memória e \_x$memória\_global\_total

Essas visualizações resumem o uso total da memória dentro do servidor.

As visualizações `_global_total_memória` e `_x$memória_global_total` têm as seguintes colunas:

* `total_allocated`

  Os bytes totais de memória alocados dentro do servidor.
