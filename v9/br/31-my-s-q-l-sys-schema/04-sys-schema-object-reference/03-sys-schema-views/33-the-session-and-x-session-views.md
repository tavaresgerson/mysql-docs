#### 30.4.3.33 As visualizações de sessão e x$session_views

Essas visualizações são semelhantes às `processlist` e `x$processlist`, mas elas filtram os processos em segundo plano para exibir apenas as sessões do usuário. Para descrições das colunas, consulte a Seção 30.4.3.22, “As visualizações processlist e x$processlist”.
