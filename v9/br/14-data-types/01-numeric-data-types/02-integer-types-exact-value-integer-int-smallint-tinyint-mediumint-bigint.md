### 13.1.2 Tipos de Inteiros (Valor Exato) - INTEGER, INT, SMALLINT, TINYINT, MEDIUMINT, BIGINT

O MySQL suporta os tipos de inteiro padrão do SQL `INTEGER` (ou `INT`) e `SMALLINT`. Como uma extensão do padrão, o MySQL também suporta os tipos de inteiro `TINYINT`, `MEDIUMINT` e `BIGINT`.
