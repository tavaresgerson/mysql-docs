A MySQL Enterprise Edition é um produto comercial. Assim como a MySQL Community Edition, a MySQL Enterprise Edition inclui o MySQL Server, um banco de dados totalmente integrado, seguro em transações, compatível com ACID, com recursos completos de commit, rollback, recuperação em caso de falha e bloqueio a nível de linha.
