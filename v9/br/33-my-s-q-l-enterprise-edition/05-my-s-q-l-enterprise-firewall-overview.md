## 32.5 Visão geral do Firewall Empresarial MySQL

A Edição Empresarial do MySQL inclui o Firewall Empresarial MySQL, um firewall de nível de aplicativo que permite que os administradores de banco de dados permitam ou negociem a execução de instruções SQL com base na correspondência com listas de permissões de padrões de instruções aceitos. Isso ajuda a proteger o MySQL Server contra ataques como injeção SQL ou tentativas de explorar aplicativos usando-os fora de suas características legítimas de carga de trabalho de consultas.

Cada conta MySQL registrada com o firewall tem sua própria lista de permissões de instruções, permitindo que a proteção seja adaptada por conta. Para uma conta específica, o firewall pode operar no modo de registro ou proteção, para treinamento nos padrões de instruções aceitos ou proteção contra instruções inaceitáveis.

Para mais informações, consulte a Seção 8.4.8, “Firewall Empresarial MySQL”.
