## 32.8 Visão geral do MySQL Enterprise Monitor

Para obter informações sobre o comportamento do MySQL Enterprise Monitor, consulte o manual do MySQL Enterprise Monitor: https://dev.mysql.com/doc/mysql-monitor/pt_BR/.

Importante

O MySQL Enterprise Monitor será descontinuado e será obsoleto a partir de 1º de janeiro de 2025.

Após essa data, o MySQL Enterprise Monitor não receberá mais atualizações de segurança, atualizações não de segurança, correções de bugs ou atualizações de conteúdo técnico online. Ele passará para o modelo de *Suporte Sustentável*.

O que esperar quando o MySQL Enterprise Monitor atingir o fim de vida (EOL):

* O MySQL cessará todas as atividades de correção de bugs para o produto
* O MySQL cessará todas as atividades de correção de segurança para o produto
* O MySQL cessará todas as atividades de desenvolvimento de novas funcionalidades para o produto

O Suporte Sustentável não inclui:

* Novas atualizações de programas, correções, alertas de segurança e atualizações de patches críticos
* Novas atualizações fiscais, legais ou regulatórias
* Novos scripts de atualização
* Certificação com novos produtos/versões de terceiros
* Compromisso de 24 horas e diretrizes de resposta para solicitações de serviço de Grau 1, conforme definido na "Seção 9 - Definições de Grau" no documento intitulado "Políticas de Suporte Técnico do Oracle"

* Correções ou atualizações anteriormente lançadas que o Oracle retirou da publicação. Os pacotes de software publicados anteriormente ou existentes permanecerão disponíveis como conteúdo arquivado.

Para o conjunto de Políticas de Suporte Técnico da Oracle, visite: <https://www.oracle.com/support/policies.html>

Para uma explicação dos diferentes modelos de suporte (como o Suporte Sustentável), visite: <https://www.oracle.com/support/lifetime-support/>
