### 10.3.7 Verifying Index Usage

Always check whether all your queries really use the indexes
that you have created in the tables. Use the
[`EXPLAIN`](explain.html "15.8.2 EXPLAIN Statement") statement, as described
in [Section 10.8.1, “Optimizing Queries with EXPLAIN”](using-explain.html "10.8.1 Optimizing Queries with EXPLAIN").
