### 27.4.2 Trigger Metadata

To obtain metadata about triggers:

* Query the `TRIGGERS` table of the `INFORMATION_SCHEMA` database. See Section 28.3.50, “The INFORMATION_SCHEMA TRIGGERS Table”.

* Use the `SHOW CREATE TRIGGER` statement. See Section 15.7.7.13, “SHOW CREATE TRIGGER Statement”.

* Use the `SHOW TRIGGERS` statement. See Section 15.7.7.41, “SHOW TRIGGERS Statement”.
