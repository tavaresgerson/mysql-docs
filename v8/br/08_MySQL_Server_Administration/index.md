# Capítulo 7 Administração do servidor MySQL

O MySQL Server (**mysqld**) é o programa principal que realiza a maior parte do trabalho em uma instalação do MySQL. Este capítulo fornece uma visão geral do MySQL Server e abrange a administração geral do servidor:

* Configuração do servidor * O diretório de dados, particularmente o esquema do sistema `mysql`

* Arquivos de registro do servidor * Gestão de vários servidores em uma única máquina
