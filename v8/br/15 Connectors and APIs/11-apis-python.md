## 31.10 API Python para MySQL

`MySQLdb` é um driver de terceiros que oferece suporte ao MySQL para Python, compatível com a versão 2.0 do Python DB API. Ele pode ser encontrado em <http://sourceforge.net/projects/mysql-python/>.

O novo componente MySQL Connector/Python oferece uma interface para a mesma API Python e está integrado ao MySQL Server e suportado pela Oracle. Consulte o MySQL Connector/Python Developer Guide para obter detalhes sobre o conector, bem como diretrizes de codificação para aplicativos Python e código Python de exemplo.
