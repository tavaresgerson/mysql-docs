## APIs Ruby para MySQL 31.11

O gem `mysql2` Ruby fornece uma API para conectar-se ao MySQL, realizar consultas e iterar por resultados; ele é destinado a suportar MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página `mysql2` no RubyGems.org ou a página do projeto no GitHub.

Para informações sobre o contexto e a sintaxe da linguagem Ruby, consulte  Linguagem de Programação Ruby.
