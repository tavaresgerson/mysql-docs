### 29.12.18 Tabelas do Cofre de Chaves do Schema de Desempenho

29.12.18.1 A tabela keyring\_component\_status

29.12.18.2 A tabela keyring\_keys

As seções a seguir descrevem as tabelas do Schema de Desempenho associadas ao chaveiro MySQL (consulte a Seção 8.4.4, “O Chaveiro MySQL”). Elas fornecem informações sobre a operação do chaveiro:

- `keyring_component_status`: Informações sobre o componente de chave de segurança em uso.

- `keyring_keys`: Metadados para chaves no chaveiro MySQL.
