## 32.9 Telemetria do MySQL

O componente de telemetria do MySQL é baseado no projeto OpenTelemetry (OTel), um framework de observabilidade de código aberto e neutro em relação aos fornecedores, que fornece um padrão comum de observabilidade. Ele permite que os usuários instrumentem suas aplicações para exportar dados de observabilidade: traços, métricas e logs, possibilitando uma maior granularidade no depuração e teste.

Para mais informações, consulte o Capítulo 35, *Telemetria*.
