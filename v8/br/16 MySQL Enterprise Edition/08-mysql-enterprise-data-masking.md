## 32.7 Visão geral da Máscara de Dados e Desidentificação do MySQL Enterprise

A Edição Empresarial do MySQL 5.7 e superior inclui a Máscara de Dados e Desidentificação do MySQL Enterprise, implementada como uma biblioteca de plugins que contém um plugin e várias funções carregáveis. A máscara de dados oculta informações sensíveis substituindo valores reais por substitutos. As funções de Máscara de Dados e Desidentificação do MySQL Enterprise permitem a máscara de dados existentes usando vários métodos, como ofuscação (removendo características identificadoras), geração de dados aleatórios formatados e substituição ou substituição de dados.

Para mais informações, consulte a Seção 8.5, “Máscara de Dados e Desidentificação do MySQL Enterprise”.
