## 32.8 Visão Geral do Monitor do MySQL Enterprise

Para obter informações sobre o comportamento do Monitor do MySQL Enterprise, consulte o manual do Monitor do MySQL Enterprise: https://dev.mysql.com/doc/mysql-monitor/pt_BR/.

Importante

O Monitor do MySQL Enterprise será descontinuado e será obsoleto a partir de 1º de janeiro de 2025.

Após essa data, o Monitor do MySQL Enterprise não receberá mais atualizações de segurança, atualizações não de segurança, correções de bugs ou atualizações de conteúdo técnico online. Ele passará para o modelo de **Suporte Sustentável**.

O que esperar quando o Monitor do MySQL Enterprise atingir o fim de vida (EOL):

* O MySQL cessará todas as atividades de correção de bugs para o produto
* O MySQL cessará todas as atividades de correção de segurança para o produto
* O MySQL cessará todas as atividades de desenvolvimento de novas funcionalidades para o produto

O Suporte Sustentável não inclui:

* Novas atualizações de programas, correções, alertas de segurança e atualizações de patches críticos
* Novas atualizações fiscais, legais ou regulatórias
* Novos scripts de atualização
* Certificação com novos produtos/versões de terceiros
* Compromisso de 24 horas e diretrizes de resposta para solicitações de serviço de Grau 1, conforme definido na "Seção 9 - Definições de Grau" no documento intitulado "Políticas de Suporte Técnico do Software Oracle"
* Correções ou atualizações publicadas anteriormente que a Oracle retirou da publicação. Os pacotes de software publicados anteriormente ou existentes permanecerão disponíveis como conteúdo arquivado.

Para o conjunto de Políticas de Suporte Técnico da Oracle, visite: <https://www.oracle.com/support/policies.html>

Para uma explicação dos diferentes modelos de suporte (como o Suporte Sustentável), visite: <https://www.oracle.com/support/lifetime-support/>

Para clientes que atualmente estão usando versões anteriores do Monitor do MySQL Enterprise, suas opções incluem:

* Use o Enterprise Manager para MySQL. Este é um produto gratuito para clientes com um contrato de suporte da Oracle válido. Para mais informações, visite Monitoramento abrangente e Gerenciamento de conformidade para bancos de dados MySQL usando o Enterprise Manager.
* Use as capacidades de monitoramento de banco de dados do serviço de Gerenciamento de banco de dados OCI e do serviço Ops Insights. Isso está disponível para MySQL HeatWave e MySQL on-premises. Para mais informações, visite Gerenciamento de banco de dados para MySQL HeatWave e MySQL on-premises e Ops Insights para HeatWave e MySQL on-premises.

### Implantações afetadas

* MySQL Enterprise Monitor Service Manager para Linux x86 (64-bit)
* MySQL Enterprise Monitor Agent para Linux x86 (64-bit)
* MySQL Enterprise Monitor Service Manager para Windows x86 (64-bit)
* MySQL Enterprise Monitor Agent para Microsoft Windows x86 (64-bit)
* MySQL Enterprise Monitor Service Manager para Mac OS X x86 (64-bit)
* MySQL Enterprise Monitor Agent para Mac OS X x86 (64-bit)
