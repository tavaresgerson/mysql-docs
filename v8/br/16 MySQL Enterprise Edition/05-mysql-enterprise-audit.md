## Visão Geral do Auditoramento do MySQL Enterprise

A Edição Empresarial do MySQL inclui o Auditoramento do MySQL Enterprise, implementado usando um plugin do servidor. O Auditoramento do MySQL Enterprise utiliza a API de Auditoria MySQL aberta para habilitar o monitoramento e registro padrão e baseado em políticas da atividade de conexão e consulta executada em servidores MySQL específicos. Projetado para atender à especificação de auditoria da Oracle, o Auditoramento do MySQL Enterprise oferece uma solução de auditoria e conformidade fácil de usar, pronta para uso, para aplicativos que são regidos por diretrizes regulatórias internas e externas.

Quando instalado, o plugin de auditoria habilita o MySQL Server a produzir um arquivo de log contendo um registro de auditoria da atividade do servidor. O conteúdo do log inclui quando os clientes se conectam e desconectam, e quais ações eles realizam enquanto estão conectados, como quais bancos de dados e tabelas eles acessam.

Para mais informações, consulte a Seção 8.4.5, “Auditoramento do MySQL Enterprise”.
