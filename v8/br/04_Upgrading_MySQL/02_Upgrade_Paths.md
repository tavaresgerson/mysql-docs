## 3.2 Caminhos de atualização

- A atualização do MySQL 5.7 para 8.0 é suportada. No entanto, a atualização só é suportada entre as versões de Disponibilidade Geral (GA). Para o MySQL 8.0, é necessário que você faça a atualização a partir de uma versão GA do MySQL 5.7 (5.7.9 ou superior). As atualizações a partir de versões não GA do MySQL 5.7 não são suportadas.

- É recomendável fazer a atualização para a versão mais recente antes de fazer a atualização para a próxima versão. Por exemplo, faça a atualização para a versão mais recente do MySQL 5.7 antes de fazer a atualização para o MySQL 8.0.

- A atualização que ignora as versões não é suportada. Por exemplo, a atualização direta do MySQL 5.6 para 8.0 não é suportada.

- Uma vez que uma série de lançamento atinja o status de Disponibilidade Geral (GA), a atualização dentro da série de lançamento (de uma versão GA para outra versão GA) é suportada. Por exemplo, a atualização de MySQL 8.0.`x` para 8.0.`y` é suportada. (A atualização envolvendo versões não GA de status de desenvolvimento não é suportada.) Salteando uma versão também é suportada. Por exemplo, a atualização de MySQL 8.0.`x` para 8.0.`z` é suportada. O MySQL 8.0.11 é a primeira versão de status GA dentro da série de lançamento do MySQL 8.0.
