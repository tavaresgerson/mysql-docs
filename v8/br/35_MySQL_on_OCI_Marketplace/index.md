# Capítulo 34 MySQL no Mercado OCI

Este capítulo descreve como implantar a Edição Empresarial do MySQL como uma Aplicação do Oracle Cloud Infrastructure (OCI) Marketplace. Este é um produto BYOL.

Nota

Para mais informações sobre o mercado OCI, consulte [Visão geral do mercado][(https://docs.cloud.oracle.com/iaas/Content/Marketplace/Concepts/marketoverview.htm)].

O aplicativo do Mercado de Edição Empresarial do MySQL é uma instância de computação do OCI, executando o Oracle Linux 7.9, com MySQL EE 8.0. A instalação do MySQL EE na imagem implantada é semelhante à instalação RPM, conforme descrito na Seção 2.5.4, “Instalando MySQL no Linux usando pacotes RPM da Oracle”.

Para mais informações sobre a Edição Empresarial do MySQL, consulte o Capítulo 32, *Edição Empresarial do MySQL*.

Para mais informações sobre a configuração avançada do MySQL, consulte o [Guia de implantação segura][(/doc/mysql-secure-deployment-guide/8.0/en/)].

Para mais informações sobre o Oracle Linux 7, consulte a documentação do Oracle Linux [(https://docs.oracle.com/en/operating-systems/oracle-linux/7/)]
