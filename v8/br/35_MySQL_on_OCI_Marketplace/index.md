Este capítulo descreve como implantar a Edição Empresarial do MySQL como uma Aplicação do Oracle Cloud Infrastructure (OCI) Marketplace. Este é um produto BYOL.

Nota

Para obter mais informações sobre o mercado OCI, consulte a seção Visão geral do mercado.

O aplicativo do Mercado de Edições Empresariais do MySQL é uma instância de computação da OCI, executando o Oracle Linux 7.9, com o MySQL EE 8.0. A instalação do MySQL EE na imagem implantada é semelhante à instalação RPM, conforme descrito na Seção 2.5.4, “Instalando o MySQL no Linux Usando Pacotes RPM da Oracle”.

Para obter mais informações sobre a Edição Empresarial do MySQL, consulte o Capítulo 32, *Edição Empresarial do MySQL*.

Para obter mais informações sobre a configuração avançada do MySQL, consulte o Guia de implantação segura.

Para obter mais informações sobre o Oracle Linux 7, consulte a documentação do Oracle Linux
