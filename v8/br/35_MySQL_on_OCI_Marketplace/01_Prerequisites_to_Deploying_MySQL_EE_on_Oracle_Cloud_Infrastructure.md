## 34.1 Pré-requisitos para implantar MySQL EE na Infraestrutura de Nuvem Oracle

São feitas as seguintes suposições:

* Você está familiarizado com a terminologia do OCI. Se você é novo no OCI, veja [Começando][(https://docs.cloud.oracle.com/iaas/Content/GSG/Concepts/baremetalintro.htm)].

* Você tem acesso a uma rede virtual de nuvem (VCN) e sub-rede configurada corretamente. Para mais informações, consulte [Rede Virtual][(https://docs.cloud.oracle.com/iaas/Content/Network/Tasks/quickstartnetworking.htm)].

* Você tem as permissões necessárias para implantar aplicativos do OCI Marketplace em um compartimento de sua assinatura. Para mais informações, consulte [Como as Políticas Funcionam][(https://docs.cloud.oracle.com/iaas/Content/Identity/Concepts/policies.htm)].
