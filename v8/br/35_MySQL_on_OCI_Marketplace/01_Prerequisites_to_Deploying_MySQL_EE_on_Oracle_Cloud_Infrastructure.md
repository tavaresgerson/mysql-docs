## 34.1 Pré-requisitos para implantar o MySQL EE na Infraestrutura de Nuvem da Oracle

As seguintes suposições são feitas:

- Você está familiarizado com a terminologia do OCI. Se você é novo no OCI, consulte Começar.

- Você tem acesso a uma rede virtual de nuvem (VCN) e sub-rede configuradas corretamente. Para obter mais informações, consulte Rede Virtual.

- Você tem as permissões necessárias para implantar aplicativos do OCI Marketplace em um compartimento da sua assinatura. Para obter mais informações, consulte Como as Políticas Funcionam.
