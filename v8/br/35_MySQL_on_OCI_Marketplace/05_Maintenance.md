## 34.5 Manutenção

Este produto é gerenciado pelo usuário, o que significa que você é responsável por atualizações e manutenção.

### Atualizando o MySQL

A instalação existente é baseada em RPM. Para atualizar o servidor MySQL, consulte a Seção 3.7, “Atualizando instalações binárias ou baseadas em pacotes do MySQL no Unix/Linux”.

Você pode usar `scp` para copiar o RPM necessário para a Instância de Computação OCI, ou copiá-lo a partir do Armazenamento de Objetos OCI, se você configurou acesso a ele. O Armazenamento de Arquivos também é uma opção. Para mais informações, consulte [Armazenamento de Arquivos e NFS][(https://docs.cloud.oracle.com/iaas/Content/File/Concepts/filestorageoverview.htm)].

### Backup e Restauração

O MySQL Enterprise Backup é a solução de backup e restauração preferida. Para mais informações, consulte a opção de fazer backup em armazenamento em nuvem.

Para obter informações sobre o MySQL Enterprise Backup, consulte Começando com o MySQL Enterprise Backup.

Para informações sobre o backup e restauração padrão do MySQL, consulte o Capítulo 9, *Backup e Recuperação*.
