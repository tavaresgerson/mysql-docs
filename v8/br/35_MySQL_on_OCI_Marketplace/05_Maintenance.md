## 34.5 Manutenção

Este produto é gerenciado pelo usuário, o que significa que você é responsável pelas atualizações e manutenção.

### Atualizando o MySQL

A instalação existente é baseada no RPM. Para atualizar o servidor MySQL, consulte a Seção 3.7, “Atualizando instalações binárias ou baseadas em pacotes do MySQL no Unix/Linux”.

Você pode usar `scp` para copiar o RPM necessário para a Instância de Computação do OCI ou copiá-lo do Armazenamento de Objetos do OCI, se você configurou o acesso a ele. O Armazenamento de Arquivos também é uma opção. Para mais informações, consulte Armazenamento de Arquivos e NFS.

### Backup e Restauração

O MySQL Enterprise Backup é a solução de backup e restauração preferida. Para obter mais informações, consulte Backup para armazenamento em nuvem.

Para obter informações sobre o MySQL Enterprise Backup, consulte Começando com o MySQL Enterprise Backup.

Para obter informações sobre o backup e restauração padrão do MySQL, consulte o Capítulo 9, *Backup e Recuperação*.
