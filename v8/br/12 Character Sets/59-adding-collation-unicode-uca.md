### 12.14.4 Adicionando uma Coletânea UCA a um Conjunto de Caracteres Unicode

Esta seção descreve como adicionar uma coletânea UCA para um conjunto de caracteres Unicode escrevendo o elemento `<collation>` dentro de uma descrição de conjunto de caracteres `<charset>` no arquivo `Index.xml` do MySQL. O procedimento descrito aqui não requer a recompilação do MySQL. Ele usa um subconjunto da especificação Locale Data Markup Language (LDML), disponível em <http://www.unicode.org/reports/tr35/>. Com este método, você não precisa definir toda a coletânea. Em vez disso, você começa com uma coletânea “base” existente e descreve a nova coletânea em termos de como ela difere da coletânea base. A tabela a seguir lista as coletâneas base dos conjuntos de caracteres Unicode para os quais coletâneas UCA podem ser definidas. Não é possível criar coletâneas UCA definidas pelo usuário para `utf16le`; não existe uma coletânea `utf16le_unicode_ci` que sirva como base para tais coletâneas.
