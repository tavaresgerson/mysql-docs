## 34.1 Pré-requisitos para Implantar o MySQL EE na Infraestrutura de Nuvem Oracle

As seguintes suposições são feitas:

* Você está familiarizado com a terminologia do OCI. Se você é novo no OCI, consulte  Começando.
* Você tem acesso a uma Rede de Nuvem Virtual (VCN) e sub-rede configuradas corretamente. Para mais informações, consulte  Redes de Nuvem Virtual.
* Você tem as permissões necessárias para implantar aplicativos do Oracle Marketplace em um compartimento de sua assinatura. Para mais informações, consulte  Como as Políticas Funcionam.
