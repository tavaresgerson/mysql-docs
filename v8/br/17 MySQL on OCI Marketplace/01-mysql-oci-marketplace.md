# Capítulo 34 MySQL no Mercado de Aplicações OCI

Este capítulo descreve como implantar a Edição Empresarial do MySQL como uma Aplicação do Mercado de Infraestrutura da Nuvem Oracle (OCI). Este é um produto BYOL.

::: info Nota

Para obter mais informações sobre o Mercado de Aplicações OCI, consulte  Visão Geral do Mercado.


:::

A Aplicação de Mercado de Aplicações MySQL Enterprise Edition é uma instância de computação OCI, executando o Oracle Linux 9.3, com o MySQL EE 8.4. A instalação do MySQL EE na imagem implantada é semelhante à instalação RPM, conforme descrito na  Seção 2.5.4, “Instalando o MySQL no Linux Usando Pacotes RPM da Oracle”.

Para obter mais informações sobre a Edição Empresarial do MySQL, consulte  Capítulo 32, *MySQL Edição Empresarial*.

Para obter mais informações sobre a configuração avançada do MySQL, consulte  Guia de Implantação Segura.

Para obter mais informações sobre o Oracle Linux 9, consulte  Documentação do Oracle Linux
