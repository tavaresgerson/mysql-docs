### 8.5.1 Componentes de mascaramento de dados versus o plugin de mascaramento de dados

Antes da versão 8.0.33, o MySQL habilitava recursos de mascaramento e desidentificação usando um plugin do lado do servidor, mas passou a usar a infraestrutura do componente no MySQL 8.0.33. A tabela a seguir compara brevemente os componentes de Mascaramento e Desidentificação de Dados do MySQL Enterprise e a biblioteca de plugins para fornecer uma visão geral de suas diferenças. Isso pode ajudá-lo a fazer a transição do plugin para os componentes.

Nota

Apenas os componentes de mascaramento de dados ou o plugin devem ser ativados de cada vez. Ativação de ambos os componentes e do plugin não é suportada e os resultados podem não ser os esperados.
