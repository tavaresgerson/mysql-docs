Para obter informações sobre os requisitos de armazenamento dos tipos de dados de string, consulte a Seção 13.7, “Requisitos de Armazenamento de Tipos de Dados”.

Para descrições de funções que operam em valores de string, consulte a Seção 14.8, “Funções e Operadores de String”.
