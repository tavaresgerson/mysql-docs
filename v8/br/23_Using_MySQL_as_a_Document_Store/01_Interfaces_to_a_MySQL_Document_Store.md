## 22.1 Interfaces para uma Armazenagem de Documentos MySQL

Para trabalhar com o MySQL como um repositório de documentos, você usa componentes dedicados e uma variedade de clientes que permitem a comunicação com o servidor MySQL para desenvolver aplicativos baseados em documentos.

- Os seguintes produtos do MySQL suportam o protocolo X e permitem que você use o X DevAPI na sua língua escolhida para desenvolver aplicativos que se comunicam com um servidor MySQL funcionando como um repositório de documentos:

  - MySQL Shell (que fornece implementações do X DevAPI em JavaScript e Python)

  - Conector/C++

  - Conector/J

  - Connector/Node.js

  - Conector/NET

  - Conector/Python

- O MySQL Shell é uma interface interativa para o MySQL que suporta modos JavaScript, Python ou SQL. Você pode usar o MySQL Shell para prototipar aplicativos, executar consultas e atualizar dados. Para instalar o MySQL Shell, você precisa seguir as instruções para fazer o download e instalar o MySQL Shell.

- Os guias de início rápido (tutoriais) neste capítulo ajudam você a começar a usar o MySQL Shell com o MySQL como um repositório de documentos.

  O guia de início rápido para JavaScript está aqui: Seção 22.3, “Guia de início rápido para JavaScript: Shell MySQL para Armazenamento de Documentos”.

  O guia de início rápido para Python está aqui: Seção 22.4, “Guia de início rápido do Python: Shell MySQL para Armazenamento de Documentos”.

- O *Guia do Usuário do MySQL Shell* no MySQL Shell 8.0 fornece informações detalhadas sobre a configuração e o uso do MySQL Shell.
