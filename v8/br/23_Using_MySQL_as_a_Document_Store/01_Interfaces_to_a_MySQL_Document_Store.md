## 22.1 Interfaces para uma Armazenadora de Documentos MySQL

Para trabalhar com MySQL como um banco de documentos, você usa componentes dedicados e uma escolha de clientes que permitem a comunicação com o servidor MySQL para desenvolver aplicativos baseados em documentos.

* Os seguintes produtos MySQL suportam o protocolo X e permitem que você use o X DevAPI no idioma escolhido para desenvolver aplicativos que se comunicam com um servidor MySQL que funciona como um banco de documentos:

+ MySQL Shell (que oferece implementações do X DevAPI em JavaScript e Python)

+ Conector/C++
+ Conector/J
+ Conector/Node.js
+ Conector/NET
+ Conector/Python
* O MySQL Shell é uma interface interativa para MySQL que suporta modos JavaScript, Python ou SQL. Você pode usar o MySQL Shell para prototipar aplicativos, executar consultas e atualizar dados. A instalação do MySQL Shell tem instruções para baixar e instalar o MySQL Shell.

* As guias de início rápido (tutoriais) neste capítulo ajudam você a começar a usar o MySQL Shell com o MySQL como um banco de documentos.

O guia de início rápido para JavaScript está aqui: Seção 22.3, “Guia de início rápido do JavaScript: MySQL Shell para Document Store”.

O guia de início rápido para Python está aqui: Seção 22.4, “Guia de início rápido do Python: MySQL Shell para Document Store”.

* O *Guia do Usuário do MySQL Shell* no MySQL Shell 8.0 fornece informações detalhadas sobre a configuração e o uso do MySQL Shell.
