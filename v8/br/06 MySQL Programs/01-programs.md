# Capítulo 6 Programas MySQL

Este capítulo fornece uma breve visão geral dos programas de linha de comando MySQL fornecidos pela Oracle Corporation. Ele também discute a sintaxe geral para especificar opções quando você executa esses programas. A maioria dos programas tem opções que são específicas para sua própria operação, mas a sintaxe de opções é semelhante para todos eles. Finalmente, o capítulo fornece descrições mais detalhadas de programas individuais, incluindo quais opções eles reconhecem.
