# Capítulo 6 Programas MySQL

Este capítulo oferece uma breve visão geral dos programas de linha de comando MySQL fornecidos pela Oracle Corporation. Também discute a sintaxe geral para especificar opções ao executar esses programas. A maioria dos programas tem opções específicas para sua própria operação, mas a sintaxe da opção é semelhante para todos eles. Por fim, o capítulo fornece descrições mais detalhadas de programas individuais, incluindo quais opções eles reconhecem.
