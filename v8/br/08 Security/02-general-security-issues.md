## 8.1 Problemas Gerais de Segurança

Esta seção descreve os problemas gerais de segurança a serem considerados e o que você pode fazer para tornar sua instalação do MySQL mais segura contra ataques ou uso indevido. Para informações específicas sobre o sistema de controle de acesso que o MySQL usa para configurar contas de usuário e verificar o acesso ao banco de dados, consulte  Seção 2.9, “Configuração e Teste Pós-Instalação”.

Para respostas a algumas perguntas frequentemente feitas sobre problemas de segurança do MySQL Server, consulte  Seção A.9, “Perguntas Frequentes do MySQL 8.4: Segurança”.
