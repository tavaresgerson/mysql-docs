### 8.4.1 Plugins de Autenticação

::: info Nota

Se você está procurando informações sobre o plugin `authentication_oci`, ele é apenas para o MySQL HeatWave Service. Consulte o [plugin `authentication_oci`](https://docs.oracle.com/en-us/iaas/mysql-database/doc/connecting-db-system.html#MYAAS-GUID-232CA959-1FDD-4AA8-A77D-0A551C881C09), no manual do *MySQL HeatWave Service*.

:::

As seções a seguir descrevem os métodos de autenticação substituíveis disponíveis no MySQL e os plugins que implementam esses métodos. Para uma discussão geral sobre o processo de autenticação, consulte a Seção 8.2.17, “Autenticação Substituível”.

O plugin de autenticação padrão é determinado conforme descrito em O Plugin de Autenticação Padrão.
