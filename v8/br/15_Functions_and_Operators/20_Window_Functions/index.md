O MySQL suporta funções de janela que, para cada linha de uma consulta, realizam um cálculo usando linhas relacionadas a essa linha. As seções a seguir discutem como usar funções de janela, incluindo descrições das cláusulas `OVER` e `WINDOW`. A primeira seção fornece descrições das funções de janela não agregadas. Para descrições das funções de janela agregadas, consulte a Seção 14.19.1, “Descrições de Funções Agregadas”.

Para obter informações sobre otimização e funções de janela, consulte a Seção 10.2.1.21, “Otimização de Funções de Janela”.
