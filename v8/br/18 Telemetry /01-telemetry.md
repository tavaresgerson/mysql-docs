# Capítulo 35 Telemetria

O projeto OpenTelemetry (OTel) é uma estrutura de observabilidade de código aberto e neutra em relação aos fornecedores, que fornece um padrão comum de observabilidade. Ele permite que os usuários instrumentem suas aplicações para exportar dados de observabilidade: traços, métricas e logs, possibilitando uma maior granularidade no depuração e teste.

::: info Nota

O suporte ao OpenTelemetry é um componente incluído na Edição Empresarial do MySQL, um produto comercial. Para saber mais sobre produtos comerciais, consulte <https://www.mysql.com/products/>.


:::
