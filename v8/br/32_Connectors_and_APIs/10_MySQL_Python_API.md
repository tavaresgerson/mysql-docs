## 31.10 API Python do MySQL

`MySQLdb` é um driver de terceiros que oferece suporte ao MySQL para Python, compatível com a versão 2.0 do Python DB API. Pode ser encontrado em <http://sourceforge.net/projects/mysql-python/>.

O novo componente MySQL Connector/Python oferece uma interface para a mesma API do Python e é integrado ao MySQL Server e é suportado pela Oracle. Consulte o Guia do Desenvolvedor do MySQL Connector/Python para obter detalhes sobre o conector, bem como diretrizes de codificação para aplicativos Python e código Python de amostra.
