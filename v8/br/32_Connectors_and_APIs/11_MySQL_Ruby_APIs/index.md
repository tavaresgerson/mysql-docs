## 31.11 APIs Ruby do MySQL

31.11.1 API MySQL/Ruby

31.11.1 A API Ruby/MySQL

O gem Ruby `mysql2` fornece uma API para conectar-se ao MySQL, realizar consultas e iterar pelos resultados; ele é projetado para suportar MySQL 5.7 e MySQL 8.0. Para mais informações, consulte a página `mysql2` no RubyGems.org ou a página do GitHub do projeto.

Para informações sobre o contexto e a sintaxe da linguagem Ruby, consulte Ruby Programming Language.
