# Capítulo 6 Programas MySQL

Este capítulo fornece uma breve visão geral dos programas de linha de comando do MySQL fornecidos pela Oracle Corporation. Também discute a sintaxe geral para especificar opções ao executar esses programas. A maioria dos programas tem opções específicas para sua própria operação, mas a sintaxe da opção é semelhante para todos eles. Por fim, o capítulo fornece descrições mais detalhadas dos programas individuais, incluindo quais opções eles reconhecem.
