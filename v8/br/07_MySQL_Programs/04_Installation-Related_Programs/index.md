## 6.4 Programas relacionados à instalação

6.4.1 comp\_err — Arquivo de Mensagem de Erro de Compilação do MySQL

6.4.2 mysql\_secure\_installation — Melhorar a segurança da instalação do MySQL

6.4.3 mysql\_ssl\_rsa\_setup — Criar arquivos SSL/RSA

6.4.4 mysql\_tzinfo\_to\_sql — Carregar as tabelas de fuso horário

6.4.5 mysql\_upgrade — Verificar e atualizar tabelas do MySQL

Os programas desta seção são usados durante a instalação ou atualização do MySQL.
