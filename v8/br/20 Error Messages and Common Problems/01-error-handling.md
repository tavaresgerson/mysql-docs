# Apêndice B Mensagens de Erro e Problemas Comuns

Este apêndice descreve os tipos de informações de erro fornecidas pelo MySQL e como obter informações sobre elas. A seção final é para solução de problemas. Ela descreve problemas e erros comuns que podem ocorrer e resoluções potenciais.
