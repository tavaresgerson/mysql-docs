### 13.1.2 Tipos Inteiros (Valor Exato) - INTEGER, INT, SMALLINT, TINYINT, MEDIUMINT, BIGINT

O MySQL suporta os tipos inteiros padrão do SQL `INTEGER` (ou `INT`) e `SMALLINT`. Como uma extensão do padrão, o MySQL também suporta os tipos inteiros `TINYINT`, `MEDIUMINT` e `BIGINT`.
