## Tipos de dados de string

Os tipos de dados de string são `CHAR`, `VARCHAR`, `BINARY`, `VARBINARY`, `BLOB`, `TEXT`, `ENUM` e `SET`.

Para informações sobre os requisitos de armazenamento dos tipos de dados de string, consulte a Seção 13.7, “Requisitos de armazenamento de tipos de dados”.

Para descrições de funções que operam em valores de string, consulte a Seção 14.8, “Funções e operadores de string”.
