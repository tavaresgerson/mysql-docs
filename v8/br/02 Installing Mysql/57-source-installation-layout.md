### 2.8.3 Layout do MySQL para a instalação de origem

Por padrão, quando você instala o MySQL depois de compilar a partir do código-fonte, a etapa de instalação instala arquivos em `/usr/local/mysql`. Os locais dos componentes sob o diretório de instalação são os mesmos que para distribuições binárias. Veja a Tabela 2.3,  Layout de Instalação do MySQL para Pacote Binário Genérico do Unix/Linux e Layout de Instalação do MySQL no Microsoft Windows. Para configurar locais de instalação diferentes dos padrões, use as opções descritas na Seção 2.8.7,  Opções de Configuração de Fonte do MySQL.
