### 2.1.3 Como obter o MySQL

Confira nossa página de downloads em \[<https://dev.mysql.com/downloads/>] (<https://dev.mysql.com/downloads/>) para informações sobre a versão atual do MySQL e para instruções de download.

Para plataformas Linux baseadas em RPM que usam o Yum como seu sistema de gerenciamento de pacotes, o MySQL pode ser instalado usando o MySQL Yum Repository.

Para plataformas Linux baseadas no Debian, o MySQL pode ser instalado usando o MySQL APT Repository.

Para plataformas SUSE Linux Enterprise Server (SLES), o MySQL pode ser instalado usando o MySQL SLES Repository.

Para obter a fonte de desenvolvimento mais recente, consulte a Seção 2.8.5, "Instalar MySQL Usando uma Árvore de Fonte de Desenvolvimento".
