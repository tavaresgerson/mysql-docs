### 2.1.3 Como obter o MySQL

Consulte nossa página de downloads em <https://dev.mysql.com/downloads/> para obter informações sobre a versão atual do MySQL e instruções de download.

Para plataformas Linux baseadas em RPM que utilizam o Yum como sistema de gerenciamento de pacotes, o MySQL pode ser instalado usando o Repositório MySQL Yum. Consulte a Seção 2.5.1, “Instalando o MySQL no Linux usando o Repositório MySQL Yum”, para obter detalhes.

Para plataformas Linux baseadas em Debian, o MySQL pode ser instalado usando o Repositório APT MySQL. Consulte a Seção 2.5.2, “Instalando o MySQL no Linux usando o Repositório APT MySQL”, para obter detalhes.

Para plataformas SUSE Linux Enterprise Server (SLES), o MySQL pode ser instalado usando o Repositório SLES MySQL. Consulte a Seção 2.5.3, “Usando o Repositório SLES MySQL”, para obter detalhes.

Para obter a versão mais recente da fonte de desenvolvimento, consulte a Seção 2.8.5, “Instalando o MySQL usando uma árvore de fonte de desenvolvimento”.
