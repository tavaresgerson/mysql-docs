### 2.8.1 Métodos de Instalação de Fontes

Existem dois métodos para instalar o MySQL a partir de fontes:

* Use uma distribuição padrão do MySQL. Para obter uma distribuição padrão, consulte a Seção 2.1.3, “Como obter o MySQL”. Para obter instruções sobre a compilação a partir de uma distribuição padrão, consulte a Seção 2.8.4, “Instalando o MySQL Usando uma Distribuição de Fonte Padrão”.

As distribuições padrão estão disponíveis como arquivos `tar` comprimidos, arquivos Zip ou pacotes RPM. Os arquivos de distribuição têm nomes na forma `mysql-VERSION.tar.gz`, `mysql-VERSION.zip` ou `mysql-VERSION.rpm`, onde *`VERSION`* é um número como `8.4.6`. Os nomes dos arquivos de distribuição de fontes podem ser distinguidos dos nomes das distribuições binárias pré-compiladas, pois os nomes das distribuições de fontes são genéricos e não incluem o nome da plataforma, enquanto os nomes das distribuições binárias incluem um nome da plataforma que indica o tipo de sistema para o qual a distribuição é destinada (por exemplo, `pc-linux-i686` ou `winx64`).

* Use uma árvore de desenvolvimento do MySQL. Para obter informações sobre a compilação a partir de uma das árvores de desenvolvimento, consulte a Seção 2.8.5, “Instalando o MySQL Usando uma Árvore de Fonte de Desenvolvimento”.
