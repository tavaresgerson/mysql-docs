## 2.4 Instalando o MySQL no macOS

Para uma lista das versões do macOS que o servidor MySQL suporta, consulte <https://www.mysql.com/support/supportedplatforms/database.html>.

O MySQL para macOS está disponível em várias formas diferentes:

* Instalador de pacote nativo, que usa o instalador nativo do macOS (DMG) para guiá-lo na instalação do MySQL. Para mais informações, consulte a Seção 2.4.2, “Instalando o MySQL no macOS Usando Pacotes Nativos”. Você pode usar o instalador de pacote com o macOS. O usuário que você usa para realizar a instalação deve ter privilégios de administrador.
* Arquivo TAR comprimido, que usa um arquivo compactado usando os comandos Unix `tar` e **gzip**. Para usar esse método, você precisa abrir uma janela do **Terminal**. Você não precisa de privilégios de administrador usando esse método; você pode instalar o servidor MySQL em qualquer lugar usando esse método. Para mais informações sobre como usar esse método, você pode usar as instruções genéricas para usar um tarball, a Seção 2.2, “Instalando o MySQL no Unix/Linux Usando Binários Genéricos”.

Além da instalação principal, o Instalador de pacote também inclui a Seção 2.4.3, “Instalando e Usando o Daemon de Inicialização do MySQL” e a Seção 2.4.4, “Instalando e Usando o Painel de Preferências do MySQL” para simplificar a gestão da sua instalação.

Para obter informações adicionais sobre como usar o MySQL no macOS, consulte a Seção 2.4.1, “Notas Gerais sobre a Instalação do MySQL no macOS”.
