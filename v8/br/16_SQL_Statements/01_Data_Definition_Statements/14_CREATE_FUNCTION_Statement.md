### 15.1.14 Declaração CREATE FUNCTION

A declaração `CREATE FUNCTION` é usada para criar funções armazenadas e funções carregáveis:

- Para obter informações sobre a criação de funções armazenadas, consulte a Seção 15.1.17, “Instruções CREATE PROCEDURE e CREATE FUNCTION”.

- Para obter informações sobre como criar funções carregáveis, consulte a Seção 15.7.4.1, “Instrução CREATE FUNCTION para Funções Carregáveis”.
