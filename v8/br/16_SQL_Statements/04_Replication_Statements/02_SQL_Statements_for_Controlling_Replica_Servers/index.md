Esta seção discute declarações para gerenciar servidores replicados. A Seção 15.4.1, “Declarações SQL para Controle de Servidores de Fonte”, discute declarações para gerenciar servidores de origem.

Além das declarações descritas aqui, `SHOW REPLICA STATUS` e `SHOW RELAYLOG EVENTS` também são usados com réplicas. Para obter informações sobre essas declarações, consulte a Seção 15.7.7.35, “Declaração SHOW REPLICA STATUS”, e a Seção 15.7.7.32, “Declaração SHOW RELAYLOG EVENTS”.
