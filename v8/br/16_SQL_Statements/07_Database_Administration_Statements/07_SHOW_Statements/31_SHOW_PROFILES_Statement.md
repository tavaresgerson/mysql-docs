#### 15.7.7.31 Mostrar perfis de declaração

```
SHOW PROFILES
```

A declaração `SHOW PROFILES`, juntamente com `SHOW PROFILE`, exibe informações de perfilagem que indicam o uso de recursos para declarações executadas durante o curso da sessão atual. Para mais informações, consulte a Seção 15.7.7.30, “Declaração SHOW PROFILE”.

Nota

As instruções `SHOW PROFILE` e `SHOW PROFILES` estão desatualizadas; espere-se que sejam removidas em uma futura versão do MySQL. Use o Gerenciador de Desempenho em vez disso; veja a Seção 29.19.1, “Profilagem de Consulta Usando o Gerenciador de Desempenho”.
