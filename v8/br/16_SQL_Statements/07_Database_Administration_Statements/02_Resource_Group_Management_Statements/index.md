O MySQL suporta a criação e a gestão de grupos de recursos e permite atribuir threads que estão em execução no servidor a grupos específicos, para que as threads sejam executadas de acordo com os recursos disponíveis para o grupo. Esta secção descreve as instruções SQL disponíveis para a gestão de grupos de recursos. Para uma discussão geral sobre a capacidade de grupos de recursos, consulte a Seção 7.1.16, “Grupos de Recursos”.
