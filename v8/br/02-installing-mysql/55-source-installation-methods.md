### 2.8.1 Métodos de instalação na fonte

Existem dois métodos para instalar o MySQL a partir da fonte:

- Para obter uma distribuição padrão, consulte a Seção 2.1.3, Como Obter o MySQL. Para instruções sobre a construção a partir de uma distribuição padrão, consulte a Seção 2.8.4, Instalar o MySQL Usando uma Distribuição de Fonte Padrão.

  As distribuições padrão estão disponíveis como arquivos **tar** compactados, arquivos Zip ou pacotes RPM. Os arquivos de distribuição têm nomes da forma `mysql-VERSION.tar.gz`, `mysql-VERSION.zip`, ou `mysql-VERSION.rpm`, onde `VERSION` é um número como `8.4.6`. Os nomes de arquivo para distribuições de origem podem ser distinguidos daqueles para distribuições binárias pré-compiladas em que os nomes de distribuição de origem são genéricos e não incluem nome de plataforma, enquanto os nomes de distribuição binária incluem um nome de plataforma indicando o tipo de sistema para o qual a distribuição é destinada (por exemplo, `pc-linux-i686` ou `winx64`).
- Use uma árvore de desenvolvimento MySQL. Para informações sobre a construção a partir de uma das árvores de desenvolvimento, consulte a Seção 2.8.5, Instalar MySQL Usando uma Árvore de Fonte de Desenvolvimento.
