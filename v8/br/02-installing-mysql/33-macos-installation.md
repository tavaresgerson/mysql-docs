## 2.4 Instalar o MySQL no macOS

Para uma lista de versões do macOS que o servidor MySQL suporta, consulte \[<https://www.mysql.com/support/supportedplatforms/database.html>] (<https://www.mysql.com/support/supportedplatforms/database.html>).

O MySQL para macOS está disponível em várias formas diferentes:

- Native Package Installer, que usa o instalador nativo do macOS (DMG) para guiá-lo na instalação do MySQL. Para mais informações, consulte a Seção 2.4.2, "Instalar o MySQL no macOS usando pacotes nativos". Você pode usar o instalador de pacotes com o macOS. O usuário que você usa para realizar a instalação deve ter privilégios de administrador.
- Arquivo TAR compactado, que usa um arquivo empacotado usando os comandos Unix `tar` e **gzip**. Para usar este método, você precisa abrir uma janela **Terminal**. Você não precisa de privilégios de administrador usando este método; você pode instalar o servidor MySQL em qualquer lugar usando este método. Para mais informações sobre o uso deste método, você pode usar as instruções genéricas para usar um tarball, Seção 2.2, Instalar MySQL em Unix / Linux Usando Binários Genéricos.

  Além da instalação do núcleo, o Instalador de Pacotes também inclui a Seção 2.4.3, Instalar e Usar o Daemon de Lançamento do MySQL e a Seção 2.4.4, Instalar e Usar o Painel de Preferências do MySQL para simplificar o gerenciamento de sua instalação.

Para obter informações adicionais sobre o uso do MySQL no macOS, consulte a Seção 2.4.1, "Notas Gerais sobre a Instalação do MySQL no macOS".
