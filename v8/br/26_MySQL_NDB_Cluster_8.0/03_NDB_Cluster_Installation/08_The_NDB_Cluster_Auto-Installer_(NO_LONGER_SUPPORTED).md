### 25.3.8 O instalador automático do NDB Cluster (já não é suportado)

Nota

Esse recurso foi removido do NDB Cluster e não é mais suportado. Consulte a Seção 25.2.4, “O que há de novo no MySQL NDB Cluster 8.0”, para obter mais informações.

O instalador de configuração gráfica baseado na web (Auto-Installer) foi removido no NDB 8.0.23 e não está mais incluído como parte da distribuição do NDB Cluster.
