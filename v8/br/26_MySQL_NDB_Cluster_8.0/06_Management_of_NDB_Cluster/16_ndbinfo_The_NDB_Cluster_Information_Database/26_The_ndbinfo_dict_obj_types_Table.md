#### 25.6.16.26 Tabela ndbinfo dict\_obj\_types

A tabela `dict_obj_types` é uma tabela estática que lista os possíveis tipos de objetos do dicionário usados no kernel NDB. Estes são os mesmos tipos definidos por `Object::Type` na API NDB.

A tabela `dict_obj_types` contém as seguintes colunas:

- `type_id`

  O ID de tipo para este tipo

- `type_name`

  O nome deste tipo
