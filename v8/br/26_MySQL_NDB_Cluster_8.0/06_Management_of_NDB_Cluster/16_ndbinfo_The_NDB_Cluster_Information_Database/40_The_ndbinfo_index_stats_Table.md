#### 25.6.16.40 A tabela ndbinfo index\_stats

A tabela `index_stats` fornece informações básicas sobre as estatísticas do índice `NDB`.

Informações mais completas sobre estatísticas de índice podem ser obtidas usando o utilitário **ndb\_index\_stat**.

A tabela `index_stats` contém as seguintes colunas:

- `index_id`

  ID do índice

- `index_version`

  Versão do índice

- `sample_version`

  Versão de amostra

##### Notas

Esta tabela foi adicionada no NDB 8.0.28.
