### 25.5.8 ndb\_delete\_all — Excluir todas as linhas de uma tabela NDB

O comando **ndb\_delete\_all** exclui todas as linhas da tabela `NDB` especificada. Em alguns casos, isso pode ser muito mais rápido do que o comando `DELETE` ou até mesmo `TRUNCATE TABLE`.

#### Uso

```
ndb_delete_all -c connection_string tbl_name -d db_name
```

Isso exclui todas as linhas da tabela chamada `tbl_name` no banco de dados chamado `db_name`. Isso é exatamente equivalente a executar `TRUNCATE db_name.tbl_name` no MySQL.
