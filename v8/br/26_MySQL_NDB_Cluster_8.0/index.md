# Capítulo 25 MySQL NDB Cluster 8.0

Este capítulo fornece informações sobre o MySQL NDB Cluster, uma versão de alta disponibilidade e alta redundância do MySQL adaptada para o ambiente de computação distribuída, utilizando o motor de armazenamento `NDB` (também conhecido como `NDBCLUSTER`) para permitir o funcionamento de vários computadores com servidores MySQL e outros softwares em um clúster. O NDB Cluster 8.0, agora disponível como uma versão de Disponibilidade Geral (GA) (a partir da versão 8.0.19), incorpora a versão 8.0 do motor de armazenamento `NDB`. Este capítulo também fornece informações específicas para o NDB Cluster 8.0 e abrange as versões até 8.0.43. O NDB Cluster 8.4 (NDB 8.4.7), baseado na versão 8.4 do motor de armazenamento NDB, também está disponível como uma versão LTS. Consulte o que há de novo no MySQL NDB Cluster 8.4 para informações sobre as diferenças do NDB 8.4 em comparação com as versões anteriores. O NDB Cluster 7.6 e o NDB Cluster 7.5, ainda disponíveis como versões de Disponibilidade Geral (GA), utilizam as versões 7.6 e 7.5 do `NDB`, respectivamente. O NDB Cluster 7.6 e 7.5 são versões anteriores de GA ainda suportadas em produção; para informações sobre o NDB Cluster 7.6, consulte o que há de novo no NDB Cluster 7.6. Para informações semelhantes sobre o NDB Cluster 7.5, consulte o que há de novo no NDB Cluster 7.5. As versões anteriores de GA do NDB Cluster 7.4 e NDB Cluster 7.3 incorporaram as versões `NDB` 7.4 e 7.3, respectivamente. *O NDB 7.4 e as séries de versões anteriores não são mais suportadas ou mantidas*. O NDB 8.0 e o NDB 8.4 são suportados em produção e são recomendados para novas implantações.
