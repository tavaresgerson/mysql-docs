### 27.3.2 Metadados de gatilho

Para obter metadados sobre gatilhos:

- Interrogue a tabela `TRIGGERS` do banco de dados `INFORMATION_SCHEMA`. Veja a Seção 28.3.45, “A tabela TRIGGERS do INFORMATION\_SCHEMA”.

- Use a declaração `SHOW CREATE TRIGGER`. Veja a Seção 15.7.7.11, “Declaração SHOW CREATE TRIGGER”.

- Use a declaração `SHOW TRIGGERS`. Veja a Seção 15.7.7.40, “Declaração SHOW TRIGGERS”.
