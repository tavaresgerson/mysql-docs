# Capítulo 32 MySQL Enterprise Edition

A Edição Empresarial do MySQL é um produto comercial. Assim como a Edição Comunitária do MySQL, a Edição Empresarial do MySQL inclui o MySQL Server, um banco de dados totalmente integrado, seguro em transações, compatível com ACID, com recursos completos de comprometimento, reversão, recuperação em caso de falha e bloqueio em nível de linha.
