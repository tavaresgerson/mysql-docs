### 2.5.2 Instalar o MySQL no Linux usando o repositório do MySQL APT

O repositório MySQL APT fornece pacotes `deb` para instalação e gerenciamento do servidor MySQL, cliente e outros componentes nas versões atuais do Debian e Ubuntu.

As instruções para usar o repositório MySQL APT estão disponíveis em Um guia rápido para usar o repositório MySQL APT.
