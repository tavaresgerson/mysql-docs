### 7.1.5 Referência de Variáveis do Sistema do Servidor

A tabela a seguir lista todas as variáveis do sistema aplicáveis no `mysqld`.

A tabela lista as opções de linha de comando (Cmd-line), as opções válidas em arquivos de configuração (Arquivo de opção), as variáveis do sistema do servidor (System Var) e as variáveis de status (Status var) em uma lista unificada, com uma indicação de onde cada opção ou variável é válida. Se uma opção do servidor definida na linha de comando ou em um arquivo de opção difere do nome da variável do sistema correspondente, o nome da variável é anotado imediatamente abaixo da opção correspondente. O escopo da variável (Var Scope) é Global, Sessão ou ambos. Consulte as descrições dos itens correspondentes para obter detalhes sobre a configuração e uso das variáveis. Quando apropriado, são fornecidos links diretos para informações adicionais sobre os itens.
