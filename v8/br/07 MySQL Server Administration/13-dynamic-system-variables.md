#### 7.1.9.2 Variaveis do sistema dinâmico

Para uma descrição dos requisitos de privilégio para a definição de variáveis do sistema, ver Seção 7.1.9.1, "Privilégios de Variaveis do Sistema".

A tabela a seguir lista todas as variáveis do sistema dinâmico aplicáveis dentro do `mysqld`.

A tabela lista o tipo de dados e o escopo de cada variável. A última coluna indica se o escopo de cada variável é Global, Sessão ou ambos. Consulte as descrições de itens correspondentes para obter detalhes sobre a configuração e uso das variáveis. Quando apropriado, são fornecidos links diretos para mais informações sobre os itens.

As variáveis que têm um tipo de string assumem um valor string. As variáveis que têm um tipo de numeric assumem um valor numérico. As variáveis que têm um tipo de boolean podem ser definidas como 0, 1, \[`ON`] ou \[`OFF`]. As variáveis marcadas como enumeration normalmente devem ser definidas como um dos valores disponíveis para a variável, mas também podem ser definidas como o número que corresponde ao valor de enumeração desejado. Para variáveis enumeradas do sistema, o primeiro valor de enumeração corresponde a 0. Isso difere do tipo de dados da tabela `ENUM` usado para colunas, para o qual o primeiro valor de enumeração corresponde a 1.
