# Capítulo 7 Administração do Servidor MySQL

O MySQL Server (`mysqld`) é o principal programa que faz a maior parte do trabalho em uma instalação MySQL. Este capítulo fornece uma visão geral do MySQL Server e abrange a administração geral do servidor:

- Configuração do servidor
- O diretório de dados, particularmente o esquema do sistema `mysql`
- Os ficheiros de registo do servidor
- Gestão de múltiplos servidores numa única máquina

Para informações adicionais sobre questões administrativas, ver também:

- Capítulo 8, "Segurança"
- Capítulo 9, "Backup e Recuperação"
- Capítulo 19, "Replicação"
