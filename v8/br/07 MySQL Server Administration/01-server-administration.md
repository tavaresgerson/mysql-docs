# Capítulo 7: Administração do Servidor MySQL

O Servidor MySQL (`mysqld`) é o principal programa que realiza a maior parte do trabalho em uma instalação do MySQL. Este capítulo fornece uma visão geral do Servidor MySQL e aborda a administração geral do servidor:

* Configuração do servidor
* O diretório de dados, particularmente o esquema do sistema `mysql`
* Arquivos de log do servidor
* Gerenciamento de múltiplos servidores em uma única máquina
