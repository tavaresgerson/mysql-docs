### 1.2.3 História do MySQL

Começamos com a intenção de usar o sistema de banco de dados `mSQL` para conectar às nossas tabelas usando nossas próprias rotinas de baixo nível (ISAM) rápidas. No entanto, após algumas testagens, chegamos à conclusão de que o `mSQL` não era rápido o suficiente ou flexível o suficiente para atender às nossas necessidades. Isso resultou em uma nova interface SQL para o nosso banco de dados, mas com uma interface de API quase idêntica à do `mSQL`. Essa API foi projetada para permitir que o código de terceiros escrito para uso com o `mSQL` fosse facilmente adaptado para uso com o MySQL.

MySQL é nomeado após a filha do cofundador Monty Widenius, My.

O nome do Delfim do MySQL (nosso logotipo) é “Sakila”, que foi escolhido de uma enorme lista de nomes sugeridos por usuários em nosso concurso “Nomeie o Delfim”. O nome vencedor foi enviado por Ambrose Twebaze, um desenvolvedor de software de código aberto da Eswatini (anteriormente Suazilândia), África. Segundo Ambrose, o nome feminino Sakila tem suas raízes no SiSwati, a língua local da Eswatini. Sakila também é o nome de uma cidade em Arusha, Tanzânia, perto do país de origem de Ambrose, Uganda.
