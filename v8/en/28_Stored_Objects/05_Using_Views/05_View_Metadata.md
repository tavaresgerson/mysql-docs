### 27.5.5 View Metadata

To obtain metadata about views:

* Query the [`VIEWS`](information-schema-views-table.html "28.3.48 The INFORMATION_SCHEMA VIEWS Table") table of the
  `INFORMATION_SCHEMA` database. See
  [Section 28.3.48, “The INFORMATION\_SCHEMA VIEWS Table”](information-schema-views-table.html "28.3.48 The INFORMATION_SCHEMA VIEWS Table").

* Use the [`SHOW CREATE VIEW`](show-create-view.html "15.7.7.13 SHOW CREATE VIEW Statement")
  statement. See [Section 15.7.7.13, “SHOW CREATE VIEW Statement”](show-create-view.html "15.7.7.13 SHOW CREATE VIEW Statement").
